# 背景画像配置ガイド

## 配置先

```
public/backgrounds/
├── bg-01.jpeg
├── bg-02.jpeg
├── ...
└── bg-21.jpeg
```

## 命名規約

- リュウドウが配布した `1 (1).jpeg` 〜 `1 (21).jpeg` を `bg-01.jpeg` 〜 `bg-21.jpeg` にリネームして配置。
- スペース・括弧を含むファイル名は `staticFile()` で問題を起こすので必ず英数ハイフンに変換する。

## リネーム・コピー手順

```bash
# 元画像があるディレクトリから
mkdir -p public/backgrounds
for i in $(seq 1 21); do
  src="元画像/1 (${i}).jpeg"
  dst="public/backgrounds/bg-$(printf '%02d' ${i}).jpeg"
  cp "${src}" "${dst}"
done
ls public/backgrounds/
```

## 画像仕様

- 寸法: ほぼ全て 2752×1536（16:9）、画像 10 だけ 1376×768
- 形式: JPEG
- Remotion 側は `objectFit: 'cover'` で 1920×1080 にフィットさせるので多少の違いは問題ない

## 加工禁止

- 画像の切り抜き・色調補正・要素追加/削除は一切行わない。
- 「背景として丸ごと使う」ことがリュウドウからの明確な指示。
- 空きスペースをキャラ配置やコンテンツエリアに割り当てるのみ。

## 備考

- エリア座標の **具体的な数値** は各 `templates/scene-XX_*.md` に記載。
- 数値は目安であり、Claude Code が背景画像を Read して最終調整して良い。
- エリアが重なる場合は、コンテンツ > キャラ > 字幕の zIndex 優先順（キャラと字幕の順は背景次第）。
