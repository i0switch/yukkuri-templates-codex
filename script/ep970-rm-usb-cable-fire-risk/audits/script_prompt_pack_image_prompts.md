各シーンの対象シーン全文を主入力にし、画像生成プロンプト内では既存キャラクター名を避けるため話者A/話者Bに匿名化した。会話全文を画像内に並べる指示は禁止し、図解・アイコン・小物・状況説明ビジュアル・概念図として生成する。実在人物・実在ブランド・既存キャラクター・実在UI・ウォーターマーク・英語ラベル・英語UIを禁止した。下部には白帯・チャット欄・テキストボックス風の余白を作らないよう指定した。image_style全件共通: 枕元の充電シーン・被覆がひびわれたケーブルのクローズアップ・PSEマーク・発熱を示す暖色警告色・白と赤の配色を基調にしたシンプルな安全診断図解。

s01〜s10の計10プロンプトを作成した。各シーンのimage_roleとcomposition_type：
- s01: 不安喚起 / 事故寸前構図
- s02: 理解補助 / チェックリスト
- s03: 不安喚起 / 失敗例シミュレーション
- s04: 理解補助 / 原因マップ
- s05: 不安喚起 / 誇張図解
- s06: 手順整理 / 手順図
- s07: 手順整理 / 証拠写真風
- s08: 手順整理 / チェックリスト
- s09: 比較 / ビフォーアフター
- s10: 手順整理 / 手順図

直投げ型プロンプト形式を採用し、Scene02のsub枠はRemotion側bulletsに任せ、main画像は長文説明を載せず字幕とキャラクターを避ける余白のある16:9素材にする方針で統一した。旧版の禁止タグ（sprite sheet、asset sheet、8枚グリッドなど）も全件で明記して排除している。
