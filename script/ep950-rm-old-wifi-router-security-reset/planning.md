## 2. 視聴者像

**具体的な人物設定**

田中さん（42歳・会社員・東京郊外在住）。
5〜7年前にプロバイダから届いたルーターをそのまま使い続けている。
Wi-Fiが繋がれば問題ないと思っており、管理画面を一度も開いたことがない。
先日、近所の人から「不審なデバイスがルーターに繋がっていた」という話を聞いてから漠然と不安になった。
でも「設定変えて繋がらなくなったら怖い」という恐怖が先に立ち、手が出せない。

## 3. 視聴者の悩み・痛みポイント

- ルーターは一度設置したら触らなくていいものだと思っている
- 「パスワード変えたことない」が恥ずかしいと感じつつ、変え方が分からない
- SSID名が「BUFFALO-XXXXXX」のまま→機種がバレると知らない
- 管理画面（192.168.1.1など）へのアクセス方法を知らない
- ファームウェア更新が必要と思ったことがない
- 「設定を変えて接続が切れたらどうしよう」という恐怖
- 家族全員のデバイスを再接続しなければならない手間への忌避感
