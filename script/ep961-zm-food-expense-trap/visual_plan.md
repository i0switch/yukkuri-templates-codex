## s01 節約してるのに増えてる
レシート合計が積み上がり、買い物かごの特売品が増えていく様子を示す導入図

sub: 安いだけで買わない / 合計額を見る / 使い切れる量を考える

## s02 特売は入口で罠になる
割引札に引き寄せられる買い物かごと、必要リスト外の商品が増える図解

sub: 必要なら得 / 不要なら出費 / リスト外は一呼吸

## s03 まとめ買い最強説を疑う
大量購入した食材が使い切れるものと傷みやすいものに分かれる比較図

sub: 保存できる物だけ多め / 葉物は少なめ / 予定なし大量買いNG

## s04 冷蔵庫の奥でお金が消える
冷蔵庫の奥に隠れた食品と、手前に期限近い食品をまとめた改善後の図解

sub: 奥に隠さない / 期限近いものは手前 / 買う前に中を見る

## s05 メニューから決めると余る
食べたいメニューから買う流れと、冷蔵庫の在庫から一品を決める流れの比較図

sub: 在庫から1品決める / 足りない物だけ買う / 新メニューは後回し

## s06 何も買うなではない
必要、予定あり、気分買いの3分類に分かれた買い物かごの図解

sub: 必要な物 / 予定がある物 / 気分買い

## s07 買い物リストが長すぎる
長すぎる願望リストが、カテゴリ別の短い買い物リストへ整理される図解

sub: 在庫を見る / カテゴリで分ける / 調整枠を残す

## s08 週1まとめ買いで失敗する理由
週1大量買いで余る食品と、保存品は週1・生鮮は少量追加する買い方の比較図

sub: 保存品は週1 / 生鮮は少量追加 / 予定変更を見込む

## s09 下げる行動は3つでいい
冷蔵庫確認、在庫から一品、リスト外予算の3ステップを示すチェックカード

sub: 買う前に冷蔵庫を見る / 在庫から1品決める / リスト外は枠を作る

## s10 NG節約はリバウンドする
極端な節約で空腹になり、反動買いする流れを注意マーク付きで示す図解

sub: 極端に削らない / 栄養を無視しない / 反動買いに注意

## s11 今週末は冷蔵庫から始める
冷蔵庫写真、在庫から一品、買い物リスト、リスト外上限の締め行動カード

sub: 冷蔵庫を撮る / 在庫で1品決める / リスト外上限を決める
