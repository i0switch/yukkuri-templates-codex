## 詳細設計補足
冒頭は「消したら戻らないかもしれない」「解約したのに請求が残る」という即時損失を置き、視聴者が自分ごと化できる入りにした。中盤では単なる説明の継続ではなく、RMはゴミ箱と復元期限、ZMは無料体験の更新日を再フックにして、ここで見落とすと損をする構造を明示した。後半は抽象論を避け、別端末で見る、二か所に逃がす、削除文言を読む、契約元を探す、証拠を残す、明細を見届ける、という手順に落とした。10シーンすべてに「視聴者の疑問」「解説役の短い訂正」「具体例または比喩」「次の行動」を置くことで、5分尺での情報密度と会話テンポを両立させた。
