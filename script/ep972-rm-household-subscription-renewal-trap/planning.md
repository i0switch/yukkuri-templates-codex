# クレカ明細に潜む家計サブスク更新トラップ planning

- episode_id: ep972-rm-household-subscription-renewal-trap
- pair: RM
- theme: 家計のサブスク自動更新が積み上がる罠と、今日できる15分整理手順
- target_duration_sec: 300
- allowed_duration_sec: 270-330
- selected_template: Scene14
- output_resolution: 1280x720
- target_dialogue_lines: 110
- audience: 少額サブスクや無料体験が増え、カード明細の確認を後回しにしがちな生活者

## 感情曲線

1. 冒頭で日常の痛みを数字や失敗で自分ごと化する。
2. 中盤で、視聴者が犯人だと思っていたものをひっくり返す。
3. 後半で、今日できる短時間手順に落として不安を行動へ変える。
4. 最後はコメントしやすいあるある共有と、一つの最終行動で終える。

## シーン設計

### s01 月580円が年6,960円に化ける
- role: intro
- scene_goal: 小さな自動更新が家計を削る痛みを作る
- sub: 月580円でも年6,960円 / 無料体験は更新日が本番 / 明細名は分かりにくい / 今日見るのは3か所

### s02 無料体験は終わった後が本番
- role: problem
- scene_goal: 無料体験から有料更新へ移る心理的罠を描く
- sub: 登録日は覚える / 終了日は忘れる / 通知メールは埋もれる / 初月無料ほど油断

### s03 明細名が知らない会社名になる罠
- role: explain
- scene_goal: カード明細名と実サービス名が一致しない問題を説明する
- sub: 明細名≠サービス名 / 決済代行名で出る / 家族利用も混ざる / 検索は金額＋日付

### s04 家族アカウントで犯人が迷子になる
- role: example
- scene_goal: 家族カードや共有アカウントの見落としを具体化する
- sub: 家族カードを確認 / 共有IDの購入履歴 / 子どものアプリ課金 / 責める前に証拠

### s05 犯人は高額請求より毎月の小銭
- role: rehook
- scene_goal: 中盤再フックとして少額反復の怖さを強調する
- sub: 高額より少額反復 / 300円×5個で月1,500円 / 年なら18,000円 / 気づきにくさが本体

### s06 明細チェックは3分類で止める
- role: solution
- scene_goal: 確認作業を継続・必要・不明の三分類にする
- sub: 必要: 使っている / 停止候補: 使ってない / 不明: 金額＋日付で検索 / 保留は1週間だけ

### s07 解約前にスクショを残す理由
- role: procedure
- scene_goal: 解約前後の証拠保存とトラブル予防を教える
- sub: 契約画面を保存 / 次回更新日を保存 / 解約完了メールを保存 / 問い合わせ先も控える

### s08 二重請求に見える一時保留もある
- role: caution
- scene_goal: 仮売上や締め日のズレを説明し、早合点を避ける
- sub: 仮売上の可能性 / 締め日のズレ / 返金処理の時差 / 不明ならカード会社へ

### s09 今日やる15分の整理手順
- role: action
- scene_goal: 視聴者が今日実行できる手順に落とす
- sub: 1: 明細を3か月表示 / 2: 同額反復を丸付け / 3: メール検索 / 4: 停止候補を解約 / 5: 来月確認

### s10 最終行動は明細を三か月だけ開く
- role: cta
- scene_goal: コメント誘導と今日の一行動で締める
- sub: 今日の行動: 三か月明細 / 探す: 同額の繰り返し / 残す: 解約スクショ / 来月: 消えたか確認
