3. **中盤（2〜3.5分）**: まとめ買いの落とし穴・食品ロスの隠れコスト → 発見
4. **中盤再フック（3.5分）**: 「じゃあ何も買うなってこと？」→ ずんだもんの暴走ボケ → 笑い
5. **後半（3.5〜4.5分）**: 食費を下げる3つの具体行動 → やる気
6. **エンディング（4.5〜5分）**: 今週末からやること1つに絞る → 安心・行動促進

## 情報ゴール
