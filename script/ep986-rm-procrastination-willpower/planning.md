# Planning: ep986-rm-procrastination-willpower

## 1. 視聴者の誤解
- 視聴者が最初に思っていそうなこと: 「自分が怠惰だから先延ばしにするんだ」「やる気が出ればできるはず」
- それが危ない/損する理由: 「気合い」で解決しようとして失敗し、自己肯定感を下げてさらに動けなくなる負のループ。
- 最後に持って帰らせる行動: 「5秒だけやる」「ハードルを極限まで下げる」仕組みを作る。

## 2. 感情曲線
- s01: 自分ごと化（「やる気が出たらやる」と言って寝転がる霊夢）
- s02-s03: 誤解の破壊（先延ばしは「性格」ではなく「脳の防御反応」）
- s04-s05: 危機感（脳内の「報酬系」が目先の楽に負け続ける末路）
- s06-s08: 解決できそう感（意志力を使わずに体を動かす「5秒ルール」と「ベビーステップ」）
- s09-s10: 今日やる行動（今すぐ「パソコンを開くだけ」をやる）

## 3. キャラ役割
- 霊夢（視聴者代表）: 「明日から本気出す」のプロ。気合で解決しようとして3分で挫折する。
- 魔理沙（解説役）: 心理学や脳科学の知見から霊夢の「気合論」を論破。脳の仕組みをハックする方法を教える。

## 4. その動画の勝ち筋
- この動画で一番強いフック: 「『やる気が出たらやる』は一生始まらない」
- 中盤の再フック: 「先延ばしは、脳が『恐怖』を感じている証拠」
- コメント欄で答えさせる一言: 「今、一番先延ばしにしてることは？」
- 最後にやらせる行動: この動画を閉じたら、その作業を「1分だけ」やる。

## 5. 構成案
- s01: 【フック】やる気は「待つ」ものではなく「出す」もの
- s02: 霊夢の気合論 vs 脳の現実
- s03: なぜ脳は「重要な作業」ほど避けたがるのか？
- s04: 【再フック】「怠惰」ではなく「恐怖」：完璧主義の罠
- s05: 意志力は「朝の洗面」で使い切っている？
- s06: 解決策1：5秒ルールで脳をジャックする
- s07: 解決策2：ハードルを「ゴミ」レベルまで下げる技術
- s08: 解決策3：ポモドーロ・テクニックの本当の使い方
- s09: 結論：今日から「気合」をゴミ箱に捨てよう
- s10: エンディング：今すぐこの動画を閉じてやるべきこと
