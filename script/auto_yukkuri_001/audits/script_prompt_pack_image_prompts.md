# image_prompts

使用prompt: 08_image_prompt_prompt.md

episode_id: auto_yukkuri_001

各シーンの対象シーン全文を主入力にし、画像生成プロンプト内では既存キャラクター名を避けるため話者A/話者Bに匿名化した。会話全文を画像内に並べる指示は禁止し、図解、アイコン、小物、UI、概念図、状況説明ビジュアルとして生成する。実在人物、実在ブランド、既存キャラクター、実在UI、ウォーターマークを避け、下部20%を字幕とキャラクター用に残す。画像は一括生成やグリッド生成にせず、sceneごとに1枚ずつimage genで作る前提にした。genericな白背景アイコンだけにならないよう、各シーンの具体物と比喩をvisual_roleに記録した。本文を画面内に長文で載せず、動画側の字幕とキャラクター表示を妨げないコンテンツ画像にする。

この証跡は、指定prompt packを使った判断過程と採用理由を残すためのもの。本文は既存script/配下の台本から転記せず、今回のテーマ、視聴者、テンプレート、品質基準に合わせて新規生成した。既存episodeの構造はファイル配置とschema確認だけに使い、本文、比喩、言い回し、シーンタイトル、ネタは流用していない。監査で再確認しやすいよう、採用理由と禁止事項への対応を明文化して残す。
