# 台本生成プラン

## 1. 入力整理
- テーマ：ふだん使ってるのに知らない物理現象 5選
- 想定尺：5分（10〜12シーン、90〜130セリフ）
- キャラペア：ZM（ずんだもんが視聴者代表 / めたんが解説、比率 ずんだもん40：めたん60）
- 使用テンプレート：`Scene04`（whiteboard-desk。タイトルライン上 + 大きな白板 + 下部グレー机）
- 仮定：
  - サブ枠なし（Scene04 LAYOUT に sub なし）→ 全シーン `sub: null`
  - subtitle は bar 字幕（白文字）。全セリフ 25文字以内
  - お題は 5項目固定。各お題 2シーン構成（導入 + 補足/具体例）で計10シーン、開幕＋締めで合計12シーン
  - 物理現象は身近・日常で観察できるものに絞る。高度数式・専門用語の連発禁止
  - 視聴者代表はずんだもん、解説はめたん。ずんだもんの「のだ／なのだ」目標 20〜40%、3連続禁止
  - めたん「〜だわ」「〜のよ」「〜ね」中心、語尾連続回避
  - ずんだもん一人称「ボク」、めたん一人称「わたくし」（characters.json 準拠）

## 2. テンプレート読取
- main_content：白板の下半分（x:100 y:180 w:1720 h:680）。各お題のメイン図解 1枚をここに置く
- sub_content：なし（Scene04 LAYOUT に sub 定義なし）
- subtitle_area：bar 字幕（x:0 y:880 w:1920 h:200）、白文字。`dialogue[].text` から表示。25字上限
- title_area：上部タイトルライン（x:100 y:60 w:1720 h:100）。章タイトル / `title_text` を入れる
- character_layout：下部机の上に2体並び。ずんだもん（左, x:180）/ めたん（右, x:1740）
- avoid_area：背景の白板の上端タイトルライン領域。subtitle bar (y:880-) 領域には main 画像を被せない

## 3. 企画角度
- 採用角度：誤解訂正型（「これただの当たり前でしょ」と思ってる現象が実は名前のある物理法則）
- 視聴者が最初に持つ疑問：物理って学校以来関係なくない？
- 一般的な誤解：物理は高校で終わった、日常は科学と無関係
- 意外な結論：毎日使ってる現象に全部名前と原理がある
- 視聴後に言える一言：「あ、これあの現象なのだ」と気づける目線が増える

## 4. 掛け合い方針
- 視聴者代表キャラの立ち位置：ずんだもん。雑な決めつけ、極端な解決策、「ボクすごい発見したのだ」ボケ担当
- 解説役の立ち位置：めたん。冷静に訂正、日常例で言い換え、2セリフで返す、3連続独演しない
- 主な勘違い：「物理って難しい」「日常と関係ない」「これただの偶然」
- 感情の流れ：軽く「えっ」→ 「ボクも知ってたのだ」と早合点 → 中盤「これも？あれも？」と発見モード → 最終「日常の見え方が変わるのだ」と納得
- 口癖の偏り対策：ずんだもん「のだ／なのだ」を強リアクション・シーン締めで使う。ただの相槌に水増ししない。3連続禁止。めたんは「のよ／だわ／ね」で分散

## 5. ボケパターン3案
1. ずんだもんが極端解決策：「全部物理で説明できるなら、ずんだもんも空飛べるのだ」 → めたんツッコミ「重力ってご存じ？」
2. ずんだもんが雑な早合点：「あ、コアンダ効果ね、知ってたのだ（実は知らない）」 → 直後の説明で間違いに気づく
3. ずんだもんが小ネタ脱線：「お風呂のお湯が排水で渦巻くやつもこれ？」 → めたん「それは別の現象、話戻すわよ」

## 6. 冒頭フック案
1. 損失型：「毎日触ってるのに名前を知らない物理現象、5つあるのだ？」
2. 誤解訂正型：「物理は学校で終わった、と思ったあなた。今日から見え方が変わるのよ」
3. あるある型：ずんだもん「シャワーの水って曲がるのだ？」→ めたん「それが今日の1つ目よ」

採用：**3 あるある寸劇フック型**（ずんだもんのキャラ立ち + 1つ目に直結する橋渡し）

## 7. 尺・密度設計
- 想定尺：約5分（300秒）
- 目標シーン数：12シーン（s01 開幕 / s02 フック先出し / s03〜s12 本編10シーン = 5項目×2 / s12 で締め兼最終行動）
- 目標本編セリフ数：100〜110セリフ（1シーン平均 8〜10セリフ）
- 1シーン平均セリフ数：9
- 中盤再フック位置：s07 冒頭（5項目中 3項目目に入る前）に「ここまで2つ。3つ目は飲み物のあのやつなのだ？」
- ずんだもん語尾「のだ／なのだ」目標：20〜40%（厳守、3連続禁止）。0% / 50%以上は修正
- めたん語尾は「のよ／だわ／ね」で分散、3連続回避

## 12. 参照動画型メモ
- 使用有無：N/A（通常リスト型）
- reference_style：—
- midpoint_rehook 予定：s07 冒頭で「ここまで2つ。3つ目は飲み物のあのやつなのだ？」
- summary 予定：s12 で「日常の見え方が変わる」を結論として置く
- 断定回避が必要な箇所：s04 飛行機の翼の揚力は「コアンダ効果のみ」と断定すると物理学的に異論があるため「応用例の一つ」と表現。s10 キャビテーションも「お風呂で同じ規模で起きる」とは断定しない

## 13. 次工程への指示
- 初稿で特に強める点：
  - s02 のフック先出しで「毎日触ってる5つ」を明確化、「目安として」明記
  - s07 中盤再フックで「ここまで2つ、3つ目は」と明示
  - s12 の最終行動を「1つ確認＋家族に話す or 翌日観察＋メモ保存」の2アクション以上で具体化
  - 各お題の橋渡し（めたん「次は…」）を s05 / s07 / s09 / s11 冒頭に必ず入れる
- 初稿で避ける点：
  - 数式・専門記号の登場（ベルヌーイ式、揚力公式など禁止）
  - 危険行為の示唆（「お風呂でキャビテーション試してみて」など禁止）
  - 実在ブランド名（メーカー、機種）→「家庭用」「市販の」と一般化
  - 断定（「飛行機が浮くのは100%コアンダのおかげ」など禁止）
  - ずんだもんの「のだ／なのだ」3連続
  - めたんの「のよ／だわ」3連続
- 必ず守るジャンル品質：
  - 全 12シーン、本編セリフ ≥ 100、1シーン平均 ≥ 8セリフ
  - 全セリフ 25文字以内
  - L3以上 リアクション 最低2回（実装目標 5回）
  - 例え話・小ネタ 最低2回（実装目標 3回）
  - 章タイトルは損失/誤解/行動/あるある/比較型のいずれか
  - 章タイトル「物理現象」テーマから浮く単語（投資・自動化・副業など）禁止
  - 最終行動 2アクション以上
  - ずんだもん「のだ／なのだ」20〜40%、3連続禁止
  - 全シーンに最低1つの数字 / 具体例 / あるある
  - シーン形式 3種類以上
