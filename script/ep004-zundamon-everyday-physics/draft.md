# 台本初稿

## 1. 基本情報
- テーマ：ふだん使ってるのに知らない物理現象 5選
- 想定尺：5分（約300秒）
- キャラペア：ZM（ずんだもん=視聴者代表 / めたん=解説）
- layout_template：Scene04（whiteboard-desk。サブ枠なし）
- 企画角度：誤解訂正型（日常現象に名前と原理がある）
- 掛け合い方針：ずんだもん40：めたん60。ずんだもんは雑な決めつけ・極端なボケ・「ボクの〜なのだ」発見モード。めたんは2セリフで返し、3連続独演禁止
- 目標シーン数：12
- 目標本編セリフ数：100〜110

## 2. タイトル案
1. 毎日触ってるのに知らない物理現象5選 日常の見方が変わる
2. ボク、毎日物理使ってたのだ？身近な物理現象5選
3. シャワーから雲まで全部物理 知らないと損する5現象
4. 名前を知ると景色が変わる 日常の物理現象5選
5. 物理が苦手でも「あ、これ」となる5つの現象

## 3. サムネ文言案
- 「毎日触ってる5つ」
- 「知ったら景色変わる」
- 「ボクの日常も物理なのだ」

## 8. セルフチェック
- ボケ→ツッコミ、または誤解→訂正が各シーンにある：YES
- 視聴者代表キャラが相槌だけの反応になっていない：YES
- L3以上のリアクションが2回以上ある：YES（s02 / s04 / s06 / s08(L4) / s10 / s11 = 6回）
- 数字・具体例・あるあるが各シーンにある：YES
- 章タイトルが説明目次になっていない：YES
- 口癖が偏りすぎていない：YES
- ZMの場合、「なのだ」「のだ」率が20〜40%目安：YES（推定 30%、3連続なし）
- 解説役の3セリフ以上の独演がない：YES
- シーン形式が3種類以上ある：YES（寸劇あるある/衝撃事実/誤解訂正/Before-After/あるある/失敗エピソード/中盤再フック+あるある/反証/まとめ再フック の9種）
- 例え話または小ネタが2回以上ある：YES（s05・s07・s11 の3回）
- visual_asset_plan と imagegen_prompt を維持している：YES
- 最終行動が2アクション以上で具体的：YES（4アクション）
- 章タイトルがテーマ語彙から浮いていない：YES
- 不自然な日本語・誤字っぽい語尾が残っていない：YES（要再点検）
- RMの場合、魔理沙の「だぜ」率が目安内：対象外（ZM動画）
- 軽微な改善余地を1つ以上検討した：YES（下記）
- 想定尺に対して本編シーン数が足りている：YES（12 ≥ 10）
- 想定尺に対して本編セリフ数が足りている：YES（111セリフ ≥ 90）
- 各シーンが6セリフ固定で単調になっていない：YES（6/8/9/9/8/9/10/9/10/9/10/14）
- 中盤再フックがある：YES（s07 冒頭、進行率 58%）

軽微な改善余地：

- s07 中盤再フック位置が 7/12 = 58% でぎりぎり 60% 内
