# ep912-rm-sleep-debt 台本プラン（Script Prompt Pack 監査済み）

## 1. 入力整理

- テーマ：睡眠負債（毎晩の睡眠不足が借金として積み重なる）
- 想定尺：3分（180秒）
- キャラペア：RM（霊夢・魔理沙）
- 使用テンプレート：Scene08（和室＋タイトル札＋白板）
- 仮定：視聴者は眠いが原因が睡眠不足だとは思っていない

## 2. テンプレート読取

- main_content：白板内（x:300 y:90 w:1320 h:760）
- sub_content：なし
- subtitle_area：下部全幅（y:880 h:200）
- title_area：上部札（x:860 y:0 w:200 h:80）
- character_layout：机上パターンB（左右キャラ）
- avoid_area：右上装飾なし、下部20%はキャラ・字幕用に空ける

## 3. 企画角度

- 採用角度：「気づかない借金」型
- 視聴者が最初に持つ疑問：なんで眠いのか
- 一般的な誤解：眠くなければ大丈夫、土日に寝だめすれば解決
- 意外な結論：眠くなくなること自体が危険のサイン
- 視聴後に言える一言：毎日30分早く寝るだけで借金を返せる

## 4. 掛け合い方針

- 霊夢：雑な理解、勘違い、ツッコミ、焦りを担当（視聴者代表）
- 魔理沙：背景・因果・具体例・結論を整理（解説役）
- 主な勘違い：眠くなければ睡眠は足りている
- 感情の流れ：意外 → ショック → 納得 → 実行決意
- 口癖の偏り対策：魔理沙の「だぜ」は30〜50%目安、連発させない

## 5. ボケパターン3案

1. 「土日に10時間寝れば一気に返せる」説（寝だめ信仰）
2. 「眠くないから大丈夫」説（感覚麻痺の誤解）
3. 「6時間で慣れた」説（慢性化の誤解）

## 6. 冒頭フック案

1. 「睡眠の借金を毎朝踏み倒してる人の話」（損失・危機感型）
2. 「眠くなくなったら逆に危ない理由」（逆説型）
3. 「6時間睡眠が2年続くと何が起きるか」（時間軸型）

## 7. 尺・密度設計

- 想定尺：180秒（3分）
- 目標シーン数：8
- 目標本編セリフ数：80
- 1シーン平均セリフ数：10
- 中盤再フック位置：s05（1時間差が15日差になる数字フック）
- 魔理沙「だぜ」目標：全セリフの30〜50%
- meta.image_style：太線フラット図解。温かみのある和色（朱・紺・黄緑）アクセント

## 9. final_action（最終行動）

- 今夜の就寝時間を確認する
- 30分早めて試す
- 枕元のスマホを消す・遠ざける
- 起床時間をチェックしてセットする

（2アクション以上：確認・試す・チェック・消す を含む）
