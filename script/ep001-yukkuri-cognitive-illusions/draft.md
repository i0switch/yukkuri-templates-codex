# 台本初稿 (ep001-yukkuri-cognitive-illusions)

## 1. 基本情報
- テーマ：人間の脳がバグる、日常の認知の錯覚7選
- 想定尺：5分前後（300秒）
- キャラペア：RM（霊夢・魔理沙）
- layout_template：Scene04
- 企画角度：誤解訂正型
- 掛け合い方針：霊夢が「自分は当てはまらない」と油断→例で自爆して焦る、魔理沙が冷静に名前付けする
- 目標シーン数：10
- 目標本編セリフ数：100前後

## 2. タイトル案
1. 認知の錯覚7選、自分の脳がバグる瞬間
2. 冷静と思った時点で詰んでる、認知バグ7選
3. あなたも毎日ハマってる、脳の罠7つ
4. 自分の判断、7割が錯覚で動いてる説
5. 人間の脳に最初から入ってるバグ7個

## 3. サムネ文言案
- 自分は冷静、で詰む

- あなたも毎日ハマってる
