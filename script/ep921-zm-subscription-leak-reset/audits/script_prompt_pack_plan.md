# planning.md

## 動画タイトル

サブスクが毎月じわっと漏れる問題を止める

## 想定視聴者

無料体験、少額サブスク、使っていない有料サービスが積み重なって毎月の支出が見えにくい人

## 視聴者の悩み

- 無料体験を入れたまま有料化している
- 月額数百円だから見逃してしまう
- 使っていないサービスが明細に残る
- 解約場所を探すのが面倒で後回しにする
- 見直しを根性でやろうとして続かない

## 冒頭の興味 / 不安

ずんだもんが少額サブスクの合計に驚き、めたんが月額は小さくても固定費として毎月残ると指摘する。

## 最後に得る納得

サブスク整理は我慢ではなく棚卸し。明細、無料体験、使った日、更新日前リマインドを見れば、残すものと止めるものを決められる。

## 感情曲線

1. 驚き: 少額が合計で大きくなる
2. 共感: 無料体験と便利さで増える理由を認める
3. 納得: 問題は金額より固定化
4. 安心: 全部解約しなくていいと分かる
5. 行動: 今日10分の棚卸しへ落ちる
