## s01 速度計測で現実を見る
スマホの速度計測画面風の抽象ビジュアルと、困った表情のない室内ルーター、弱い電波を表す波形

sub: 昼と夜で速度を比べる / 回線だけを犯人にしない / まず家の中を疑う

## s02 置き場所だけで詰む
部屋の隅や棚の奥で電波が遮られるルーターと、中央付近に移動した改善比較図

sub: 床置きは避ける / 棚の奥に入れない / 金属の近くを避ける

## s03 死角は部屋の形で生まれる
家の簡易間取りに電波が強い場所と弱い場所を色分けしたヒートマップ風図解

sub: 玄関付近 / 寝室 / いつも遅い席

## s04 隣の電波がぶつかる
集合住宅で複数のWi-Fi波形が重なり合い、混雑している様子を抽象化した図解

sub: 夜だけ遅い / 集合住宅で多い / 自動チャンネルを確認

## s05 熱でルーターは弱る
熱がこもった棚の中のルーターと、風通しの良い場所へ移した比較イメージ

sub: 熱い棚に入れない / 布をかぶせない / 通気口を空ける

## s06 全部直しても爆速とは限らない
契約速度、実測速度、家の中の損失を分けて示すメーター風図解

sub: 契約以上にはならない / 損失を減らすのが目的 / 測定で差を見る

## s07 5GHzと2.4GHzは役割が違う
5GHzは近距離高速、2.4GHzは遠距離安定として部屋の距離別に示す図解

sub: 近くは5GHz / 遠くは2.4GHz / 壁が多い場所は注意

## s08 古い機器が足を引っ張る
古いルーターのチェック項目、年数、再起動頻度、古い規格を示す診断カード風ビジュアル

sub: 5年以上使っている / 再起動が増えた / 古い規格のまま

## s09 今日の3つだけ見る
スマホ測定、ルーター移動、通気確認の3ステップを大きく示すチェックリスト

sub: 速度を3か所で測る / ルーターを見通し良く置く / 熱と通気を確認

## s10 やりがちなNGを避ける
NGマーク付きで布をかぶせたルーター、適当に置いた中継機、コードだらけの棚を示す注意図

sub: 布で隠さない / 中継機を適当に増やさない / 設定をメモなしで変えない

## s11 まずルーターを外に出す
10分タイマー、速度測定、ルーターを棚から出す、通気確認を並べた締めの行動カード

sub: 10分だけ確認 / 棚の奥から出す / 測ってから相談
