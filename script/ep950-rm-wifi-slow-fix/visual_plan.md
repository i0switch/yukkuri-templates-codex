スマートフォンの速度計測アプリに「2Mbps」という数字が表示されている衝撃の画面

sub: 光回線でも遅くなる原因がある / 問題はルーター周辺 / 自分で直せる可能性あり

## s02 棚に隠すと電波が消える
棚の中に閉じ込められたルーターと、その周囲で電波がつぶされているイメージ図

sub: 棚の中・床置きはNG / 高い位置・壁から離す / 通気確保で熱対策

## s03 電波を遮る意外な障害物
コンクリート壁と電子レンジの近くに置かれたルーターから電波が弱くなるイメージ

sub: コンクリートは電波の天敵 / 電子レンジ近くは厳禁 / 三重苦を避ける

## s04 隣の家の電波が犯人になる話
複数のルーターから重なり合う電波が飛んでいる俯瞰イメージ。チャンネルが渋滞している概念図

sub: 同じチャンネルが渋滞原因 / アナライザーアプリで確認 / 空きチャンネルに変更

## s05 再起動しても直らない本当の理由
熱がこもって電波が不安定になっているルーターのイメージ。放熱路を塞ぐ物が乗っている

sub: 上に物を置かない / 壁から10cm以上離す / 通気で安定化

## s06 全部直したら光回線より速くなる？
高速道路と合流地点の比較で「回線は道路・ルーターは合流地点」を示す概念図

sub: 回線速度が上限 / 多くは半分以下しか出ていない / 環境整備で倍速感も現実的

## s07 5GHzと2.4GHz、どっちを選ぶか
5GHzと2.4GHzの電波到達範囲の違いを示す図解。近くは速い・遠くは広いイメージ

sub: 近くは5GHz（速い） / 遠くは2.4GHz（届く） / 名前に5Gが入ってるほう
