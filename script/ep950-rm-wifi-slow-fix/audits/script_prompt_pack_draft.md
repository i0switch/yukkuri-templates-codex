# s01: 速度を計測したら衝撃だった

霊夢「2Mbpsって……Youtubeの動画すら普通に見られない速度じゃない」

霊夢「ボトルネックって、高速道路の合流地点みたいな？」
魔理沙「その通りだぜ。今日はその原因と今日から直せる対処を3つ教えるぜ」

# s02: 棚に隠すと電波が消える

霊夢「うちのルーター、棚の中に隠してるんだけど。コードが見えてると生活感出るじゃない」

霊夢「えっそんなに悪いの？見た目優先でいけないの？」
魔理沙「電波は直進性が高くて、壁や家具で急激に弱まる。棚の中は最悪の環境だぜ」
