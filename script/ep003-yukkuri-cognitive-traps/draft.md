# 台本初稿

## 1. 基本情報
- テーマ：日常に潜む“認知の罠” 5選
- 想定尺：5分（約300秒）
- キャラペア：RM（霊夢=視聴者代表 / 魔理沙=解説）
- layout_template：Scene12（classroom-bubbles。サブ枠なし）
- 企画角度：誤解訂正型
- 掛け合い方針：霊夢40：魔理沙60。霊夢は雑な決めつけ・極端な解決策・怖がりボケ。魔理沙は2セリフ以内で返し、3連続独演禁止
- 目標シーン数：12
- 目標本編セリフ数：100〜110

## 2. タイトル案
1. 全員が引っかかる「認知の罠」5選 直感を1個ずらす習慣
2. 気づかず判断ミスしてる罠5選 賢くても引っかかる仕組みの話
3. 自分は冷静、と思った人ほど危ない 認知の罠5選
4. 直感をやめると静かに勝てる 認知の罠5選
5. 1日数十回かかる罠 認知バイアス5選を仕組みで防ぐ

## 3. サムネ文言案
- 「全員引っかかる」
- 「直感は危ない」
- 「1個ずらすだけ」

## 8. セルフチェック
- ボケ→ツッコミ、または誤解→訂正が各シーンにある：YES
- 視聴者代表キャラが相槌だけの反応になっていない：YES
- L3以上のリアクションが2回以上ある：YES（s02 / s06 / s09×2 / s11 = 5回）
- 数字・具体例・あるあるが各シーンにある：YES
- 章タイトルが説明目次になっていない：YES（フック型12種）
- 口癖が偏りすぎていない：YES
- ZMの場合、「なのだ」「のだ」率が20〜40%目安：対象外（RM動画）
- 解説役の3セリフ以上の独演がない：YES
- シーン形式が3種類以上ある：YES（寸劇あるある/衝撃事実/誤解訂正/手順/あるある/Before-After/中盤再フック+あるある/反証/失敗エピソード/まとめ再フック の10種）
- 例え話または小ネタが2回以上ある：YES（s05・s09・s11 の3回）
- visual_asset_plan と imagegen_prompt を維持している：YES
- 最終行動が2アクション以上で具体的：YES（4アクション）
- 章タイトルがテーマ語彙から浮いていない：YES（投資・自動化・副業など外単語なし）
- 不自然な日本語・誤字っぽい語尾が残っていない：YES（要再点検）
- RMの場合、魔理沙の「だぜ」率が目安内：YES（推定 50%、3連続なし）
- 軽微な改善余地を1つ以上検討した：YES（下記参照）
- 想定尺に対して本編シーン数が足りている：YES（12シーン ≥ 10）
- 想定尺に対して本編セリフ数が足りている：YES（111セリフ ≥ 90）
- 各シーンが6セリフ固定で単調になっていない：YES（6/8/9/9/8/9/10/9/10/9/10/14 で変化あり）
- 中盤再フックがある：YES（s07 冒頭に midpoint_rehook）

軽微な改善余地：
- s01 が6セリフでやや短い。寸劇感はあるが情報量に余裕。1〜2セリフ追加余地あり（次工程で検討）
- s09 で「フォアラー効果」を最後にだけ呼称する形にしているが、専門用語の落とし所として妥当か audit で再点検
- s02 タイトル「全員が引っかかる罠が5つある」が説明寄りに見える可能性。フック性をもう一段上げる余地あり
