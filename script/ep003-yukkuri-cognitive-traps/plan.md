# 台本生成プラン

## 1. 入力整理
- テーマ：日常に潜む“認知の罠” 5選
- 想定尺：5分（10〜12シーン、90〜130セリフ）
- キャラペア：RM（霊夢が視聴者代表 / 魔理沙が解説、比率 霊夢40：魔理沙60）
- 使用テンプレート：`Scene12`（classroom-bubbles。教室背景＋左上吹き出しタイトル＋下部右側に字幕吹き出し overlay）
- 仮定：
  - サブ枠なし（Scene12 LAYOUT に sub なし）→ 全シーン `sub: null`
  - subtitle は overlay ラベルのみ。全セリフ 25文字以内
  - お題は 5項目固定。各お題 2シーン構成（導入 + 補足/具体例）で計10シーン、開幕＋締めで合計12シーン
  - 「認知の罠」は心理学用語をジャンル一般化した呼称として使う。医学的断定は避け、「目安」「例として」「場合がある」で逃げる
  - 視聴者代表は霊夢、解説は魔理沙。霊夢の「だぜ」混入は禁止。魔理沙の「だぜ」目標 30〜60%、3連続禁止

## 2. テンプレート読取
- main_content：教室ホワイトボード内（x:130 y:140 w:1660 h:600）。各お題のメイン図解 1枚をここに置く
- sub_content：なし（Scene12 LAYOUT に sub 定義なし）
- subtitle_area：overlay 字幕吹き出し（x:620 y:880 w:1260 h:160）。`dialogue[].text` から表示。25文字上限
- title_area：左上吹き出し（x:40 y:10 w:780 h:80）。章タイトル / `title_text` を入れる
- character_layout：下部左寄せ 2体並び。霊夢（左, x:170）/ 魔理沙（右, x:490）
- avoid_area：背景の吹き出し位置外。中央上部の白板領域は main 画像専用、字幕や追加文字は重ねない

## 3. 企画角度
- 採用角度：誤解訂正型（「自分は冷静に判断してる」と思い込む視聴者の前提を逐項目崩していく）
- 視聴者が最初に持つ疑問：自分はバイアスかかってるって自覚できる？
- 一般的な誤解：気をつければバイアスは避けられる／賢ければ引っかからない
- 意外な結論：知ってても避けられない場面が多い。回避するには仕組みで対処
- 視聴後に言える一言：「直感じゃなくて1個ずらして判断してみよ」

## 4. 掛け合い方針
- 視聴者代表キャラの立ち位置：霊夢。雑な決めつけ、極端な解決策、勘違い、怖がり、ボケ担当
- 解説役の立ち位置：魔理沙。日常例で言い換え、2セリフで返す、3連続独演しない
- 主な勘違い：「気合で意識すれば防げる」「賢い人は引っかからない」「占いはアホが信じる」
- 感情の流れ：軽く驚き → ちょい焦り → 中盤「でも回避できなくない？」と諦め → 最終「仕組みでやればいい」と納得
- 口癖の偏り対策：魔理沙「だぜ」3連続禁止、強調セリフでは外す。霊夢「〜のね」「〜じゃない」「〜かしら」で分散

## 5. ボケパターン3案
1. 霊夢が極端解決策：「全部疑えばいいじゃない、買い物も全部やめるとか」 → 魔理沙ツッコミ「日常崩れるだろ」
2. 霊夢が雑な自己診断：「私は冷静だから引っかからないわ」 → 直後のシーンで自分が引っかかる例に巻き込まれる
3. 霊夢が小ネタ脱線：「占いって全部当たって見えるってこと？じゃあ私の星占いも…」 → 魔理沙「話戻すぞ」

## 6. 冒頭フック案
1. 損失型：「気づかないうちに、毎日数回は判断ミスしてるかもしれないぜ」（魔理沙が冒頭で先出し）
2. 誤解訂正型：「自分は引っかからない、って思った人ほど危ないんだぜ」
3. 寸劇フック型：霊夢「ネット記事の見出しだけで全部わかるわ」 → 魔理沙「待て待て、それが今日のテーマだぜ」

採用：**3 寸劇フック型**（霊夢のキャラ立ちと本題への橋渡しが両立、list-type 動画② と同型）

## 7. 尺・密度設計
- 想定尺：約5分（300秒）
- 目標シーン数：12シーン（s01 開幕 / s02 フック先出し / s03〜s12 本編10シーン = 5項目×2 / 末尾は s12 で締め兼最終行動）
  - 細分化：s03-04 確証バイアス / s05 中盤再フック含む（ゲシュタルト崩壊導入）/ s06 ゲシュタルト崩壊補足 / s07-08 カクテルパーティ効果 / s09-10 フォアラー効果 / s11 プライミング1シーン圧縮 / s12 まとめ＋最終行動
- 目標本編セリフ数：100〜110セリフ（1シーン平均 8〜10セリフ）
- 1シーン平均セリフ数：9
- 中盤再フック位置：s07 冒頭（5項目中 3項目目に入る前）に「ここまで2つ。3つ目はもっとあるあるだぜ」と置く
- 魔理沙語尾目標：「だぜ」30〜60%（厳守、3連続禁止）。霊夢は「のね」「じゃない」「かしら」で分散
- ZM 語尾項目：N/A（RM 動画）

## 13. 次工程への指示
- 初稿で特に強める点：
  - s02 のフック先出しで「全員引っかかる」をはっきり数字付き（「目安として1日数十回」）で言わせる
  - s07 中盤再フックで離脱対策を明確にする
  - s12 の最終行動を「1個選んで保存／予定化／確認」の2アクション以上で具体化
  - 各お題の橋渡し（魔理沙「お次は…だぜ」相当）を s05 / s07 / s09 / s11 冒頭に必ず入れる
- 初稿で避ける点：
  - 心理学用語の連発（「ゲシュタルト」「フォアラー」を3連発させない、すぐ日常語に翻訳）
  - 実在ブランド名（カフェ系、占い系、診断アプリ系）→「カフェ系のお店で」「診断系のアプリで」と一般化
  - 医学的断定（「うつ」「PTSD」言及禁止）
  - 魔理沙の「だぜ」3連続
  - 各シーンが Q&A だけで進む構造
- 必ず守るジャンル品質：
  - 全 12シーン、本編セリフ ≥ 100、1シーン平均 ≥ 8セリフ
  - 全セリフ 25文字以内
  - L3以上 リアクション 最低2回（実装目標 4回）
  - 例え話・小ネタ 最低2回（実装目標 3回）
  - 章タイトルは損失型 / 誤解訂正型 / 行動型 / あるある型 / 比較型のいずれかを各シーンで明示
  - 章タイトル「認知の罠」テーマから浮く単語（投資・自動化・副業・婚活など）禁止
  - 最終行動 2アクション以上（確認＋選ぶ＋保存／予定化）
  - 魔理沙「だぜ」30〜60%、3連続禁止
  - 全シーンに最低1つの数字 / 具体例 / あるある
  - シーン形式 3種類以上（実装：寸劇あるある / 衝撃事実 / 誤解訂正 / 手順 / Before/After / あるある / 反証 / 失敗エピソード / まとめ再フック の9種類使用）
