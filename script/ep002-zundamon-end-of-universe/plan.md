# 台本生成プラン (ep002-zundamon-end-of-universe)

## 1. 入力整理
- テーマ：実はまだ決着してない、宇宙の終わり方
- 想定尺：約5分（300秒前後）
- キャラペア：ZM（ずんだもん・四国めたん）
- 使用テンプレート：Scene16（sf-3stack-dark）
- 仮定：4つの主要仮説（ビッグリップ／ビッグクランチ／ビッグフリーズ／ビッグバウンス）を比較し、現代観測（ハッブル定数）の食い違いも触れる、知的ミステリー寄りのサイエンス問題提起。
- アーキタイプ：science-mystery-type（5分用に圧縮、章2〜3個＋オープニング＋エンディング）

## 2. テンプレート読取（Scene16）
- main_content：中央パネル（1780x720）。仮説図解、宇宙の比喩、データグラフを置く中心エリア。
- sub_content：なし
- subtitle_area：下部 overlay 帯（1780x170）。`dialogue[].text` をここに表示、白文字。
- title_area：上部帯（1780x100）。各章タイトル。
- character_layout：左下（ずんだもん、x=70）／右下（めたん、x=1850）。画面端ギリギリの配置。
- avoid_area：上下のキャラと帯エリア。main の中央 1780x720 を使う。

## 3. 企画角度
- 採用角度：誤解訂正型（「宇宙は永遠」と思いがちだが、4仮説のどれも未決着）
- 視聴者が最初に持つ疑問：「宇宙って終わりあるの？」
- 一般的な誤解：「ビッグバンの逆＝ビッグクランチで決まり」
- 意外な結論：観測ではビッグフリーズが現有力、ただしハッブル定数で揉めてる
- 視聴後に言える一言：「宇宙の終わり方は人類まだ決められない」

## 4. 掛け合い方針
- 視聴者代表キャラの立ち位置：ずんだもんが宇宙のスケールに対してずんだ餅でボケる、雑な比喩、極端な反応
- 解説役の立ち位置：めたんが冷静に4仮説を整理、断定回避を言語化
- 主な勘違い：「宇宙は永遠」「ビッグバンの逆で潰れて終わる」「太陽が消える日＝宇宙の終わり」
- 感情の流れ：油断 → 驚き（4仮説もある） → 焦り（決着してない） → 知的好奇心 → 余韻（夜空を見たくなる）
- 口癖の偏り対策：ずんだもんの「のだ」「なのだ」は20〜40%目安、強リアクション・章締めで優先使用、相槌は「だね」「マジで？」「やばい」を混ぜる

## 5. ボケパターン3案
1. ずんだ餅の比喩：「宇宙の終わりってずんだ餅の潰し方みたいなのだ」
2. 極端な解決策：「人類が観測やめたら宇宙も終わるのだ？」
3. 自己中心的解釈：「私が寝ると宇宙も終わるのだ」

## 6. 冒頭フック案
1. 数字型：「宇宙の終わり方、まだ4仮説で揉めてる」
2. 誤解訂正型：「宇宙は永遠、と思った人ほど読んでほしいのだ」
3. ボケ→学術型：「ずんだ餅と宇宙は4種類で揃うのだ」

採用：3（ずんだ餅ボケ→4仮説本題）を主軸、終盤で回収する伏線として使う

## 7. 尺・密度設計
- 想定尺：5分前後（300秒）
- 目標シーン数：10シーン
- 目標本編セリフ数：100前後（各シーン10±2）
- 1シーン平均セリフ数：10
- 中盤再フック位置：s05（5/10 = 50%地点）
- ずんだもん語尾目標：「のだ」「なのだ」20〜40%（ずんだもん発話のうち）
- めたん語尾：「わ」「のよ」「ね」を3割以下、3連続させない

## 13. 次工程への指示
- 初稿で特に強める点：ずんだもんのずんだ餅ボケ→めたんの「比較スケールがバグってる」ツッコミの繰り返し（冒頭と終盤で対）
