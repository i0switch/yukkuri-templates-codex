# 改善版台本プロンプト 適用結果

## 1. 生成前の企画メモ
- 採用した企画角度：誤解訂正型
- 想定視聴者：雑学動画を流し見する一般視聴者
- 視聴者が最初に持つ疑問：知ってるようで説明できない身近な現象って、結局なぜ起きるの？
- 一般的な誤解：全部「なんとなくそうなる」で片づけている
- この動画の意外な結論：日常の違和感は、体・光・水・音・電気の基本ルールでかなり説明できる
- 動画のゴール：21個の疑問を短く理解し、1つは誰かに話したくなる状態にする

## 2. 動画メタ
- 動画尺目安：3分
- 解説レベル：一般向け初級
- 雰囲気：軽快、ややコミカル、でも説明は端的
- 視聴後に言えるようになる一言：身近な不思議は、だいたい仕組みで説明できる

## 7. 台本本文

### シーン1：冒頭フック
- 目的：視聴を止めて、一気見の価値を伝える
- 画面・演出：テンポよく疑問ワードを3連打
- テロップ：説明できる？
- 必要素材：疑問ワードの大テロップ、テンポ速めの切り替え

ずんだもん：知ってるのに説明できない現象、多すぎるのだ  
めたん：しかも理由を知ると、ちょっと気持ちいいのよ  
ずんだもん：今日はそれを21個まとめて回収するのだ  
めたん：流し見でも拾えるように短くいくわ  

- ミニオチ／小結論：分からないを、気持ちよく減らす回

### シーン2：体の反応編
- 目的：最初の納得を早めに出す
- 画面・演出：あくび、くしゃみ、涙のアイコンを順に表示
- テロップ：体は勝手に最適化
- 必要素材：人物シルエット、呼吸、目、鼻の記号図

ずんだもん：あくびが伝染るの、地味に不思議なのだ  
めたん：脳が相手の状態をまねしやすいからですわ  
ずんだもん：くしゃみ連発も、鼻の掃除を念入りにしてるのだ？  
めたん：だいたいそうね、体は思ったより合理的なのよ  

- ミニオチ／小結論：体の変な動きにも、だいたい目的がある

### シーン3：光と色編
- 目的：視覚的に分かりやすい山場を作る
- 画面・演出：空、夕焼け、虹をフラット画像で切り替え
- テロップ：色は空気と光の仕事
- 必要素材：青空、夕焼け、虹の簡略図

ずんだもん：空が青いのも、夕焼けが赤いのもセットで知りたいのだ  
めたん：短い色は散りやすく、長い色は最後まで残りやすいのよ  
ずんだもん：虹まで出てくると、急に理科室感が強いのだ  
めたん：でも要は光が曲がって分かれてるだけ、そこまで怖くないわ  

- ミニオチ／小結論：空の見え方は、天気より光の通り道が主役

### シーン4：水と温度編
- 目的：見た目と実際のズレで中盤の転換を作る
- 画面・演出：氷、水滴、湯気を並べる
- テロップ：見た目に騙される
- 必要素材：コップ、氷、やかん、汗の記号図

ずんだもん：冷たいコップの外に水がつくの、あれ中から漏れてる気分なのだ  
めたん：漏れてないわ、空気中の水分が外で集まってるだけよ  
ずんだもん：湯気も、見えてる時点で実は気体じゃないのだ？  
めたん：そこが面白いの、見えてる白は細かい水滴なのよ  

- ミニオチ／小結論：温度まわりは、見えるものほど本体じゃないことがある

### シーン5：音と感覚編
- 目的：身近さを維持しつつ少し知的にする
- 画面・演出：駅、部屋、イヤホンの3カット
- テロップ：耳と脳の補正
- 必要素材：音波線、耳アイコン、室内外比較図

ずんだもん：静かな部屋だと逆に耳鳴りっぽく感じるのだ  
めたん：脳が小さい音まで拾おうとするからですわ  
ずんだもん：人の声だけ妙に聞こえる時があるのも、そのせいなのだ？  
めたん：ええ、耳より脳の選別が強い場面も多いのよ  

- ミニオチ／小結論：聞こえ方は、耳だけでなく脳の編集込み

### シーン6：中盤整理
- 目的：離脱防止用に法則をまとめる
- 画面・演出：ここまでの疑問を一覧で再表示
- テロップ：共通ルール
- 必要素材：一覧カード、矢印、まとめ図

ずんだもん：ここまででも、世の中わりと説明可能なのだ  
めたん：体の最適化、光の散り方、温度差、この3つが軸ね  
ずんだもん：でもまだ静電気みたいな痛いやつが残ってるのだ  
めたん：次はそこ、厄介だけど理屈は単純よ  

- ミニオチ／小結論：複雑そうでも、裏のルールは案外少ない

### シーン7：摩擦と電気編
- 目的：終盤の印象を強くする
- 画面・演出：冬服、ドアノブ、髪の逆立ちを表示
- テロップ：冬だけ痛い理由
- 必要素材：静電気アイコン、乾燥マーク、衣服の摩擦図

ずんだもん：静電気って、冬だけ本気を出しすぎなのだ  
めたん：乾燥して電気が逃げにくいから、一気に来るのよ  
ずんだもん：つまりドアノブは悪人じゃないのだ？  
めたん：悪いのは乾いた空気と油断ね。みんなは冬の静電気、怖い派？  

- ミニオチ／小結論：痛いのは突然でも、原因はかなり地味

### シーン8：まとめ
- 目的：満足感を出してコメント導線につなぐ
- 画面・演出：21個一覧を一瞬だけ見せて締める
- テロップ：次の疑問募集
- 必要素材：一覧カード、コメント誘導テロップ

ずんだもん：今日だけで、日常が少しだけ透けて見えたのだ  
めたん：疑問が仕組みに変わると、景色の見え方も変わるのよ  
ずんだもん：次の疑問もコメントで集めたいのだ  
めたん：採用しやすいので助かりますわ  

- ミニオチ／小結論：分かると、地味な現象ほど面白い
