# s01: 旅行前の油断

- pattern: B

霊夢「スマホ100%なら余裕でしょ。朝の私、かなり強いよ」
魔理沙「駅まで40分で半分削れる日もあるぜ」
霊夢「そんなに減る？地図だけ見てるつもりなのに」
魔理沙「地図、通知、写真、決済が同時に動くんだ」
霊夢「スマホが小さい従業員みたい。しかも旅行初日から残業だね」
魔理沙「しかも休憩なしで働かせるからな」

# s02: 予備の落とし穴

- pattern: D

霊夢「モバイルバッテリー入れたら勝ち。ここで勝利宣言していい？」
魔理沙「ケーブルが違うとただの重りだぜ」
霊夢「旅行カバンに石を入れてたのか。急に修行感が出た」
魔理沙「だから前日に1回つないで確認する」
霊夢「充電器の面接みたいだね。合格発表まで見届ける」
魔理沙「落ちたやつは家に置いていく」

# s03: 今日やること

- pattern: C

霊夢「結局なにからやればいい？準備リストが増えると、私はそっと閉じるよ」
魔理沙「正直、全部やると面倒なんだよな」
霊夢「解説役が詰まった。そこは自信満々で来てほしかった」
魔理沙「だから3つだけ。充電、ケーブル、紙メモ」
霊夢「紙メモは古いけど強い。電池がなくても働くのえらい」
魔理沙「電池が切れても読めるからな」
