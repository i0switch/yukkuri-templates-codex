# シーンレイアウト判断指針

Claude Code が各背景画像を見て、キャラ位置・コンテンツエリア・字幕エリアを決めるときの判断基準。

## 基本原則

1. **背景画像の空きスペース（ほぼ均一色のエリア）を探す**
2. **背景の重要装飾要素（イラスト部分、枠、機材など）を覆わない**
3. **キャラを先に配置**してから、残った空きエリアをコンテンツ/字幕に割り当てる
4. **字幕エリアは画面下部が基本**。ただし背景が下部字幕を明示的に用意している場合（黒帯・白帯）はそれに従う

## キャラ配置パターン

### パターン A: 両端下（標準）
最も多いパターン。字幕帯の左右端にキャラが被るように配置。

```
┌────────────────────────┐
│                        │
│   [コンテンツ]         │
│                        │
├──────────────────────┤
│[C]  [字幕帯]     [C] │
└────────────────────────┘
```

**座標目安:**
- leftChar: (140, 920), scale 0.55
- rightChar: (1780, 920), scale 0.55

**適用シーン:** 02, 03, 05, 06, 07, 10, 13, 14, 15, 16, 17, 21, 22

### パターン B: 机の上
背景に机・カウンターがあるシーン。顔だけを机の上に出す感じ。

```
┌────────────────────────┐
│                        │
│   [コンテンツ]         │
│                        │
│═══════════════[机]══════│
│ [C]                [C] │
└────────────────────────┘
```

**座標目安:**
- leftChar: (180, 960), scale 0.55
- rightChar: (1740, 960), scale 0.55

**適用シーン:** 04, 08, 11, 18, 19, 20

### パターン C: 中央下 2 体並び
字幕が吹き出し型で、キャラがその隣に並ぶ。

```
┌────────────────────────┐
│                        │
│   [コンテンツ]         │
│                        │
│                        │
│  [C][C] ⟨◇字幕吹出⟩   │
└────────────────────────┘
```

**適用シーン:** 12（タイトル吹出が左上、字幕吹出が右下）

### パターン D: コンテンツ枠の外側下端
コンテンツ枠がある程度余白のある中央配置のとき。

**適用シーン:** 01, 09

## エリアサイズ判断テーブル

### メインコンテンツエリア

| 背景タイプ | 座標目安 |
|---|---|
| 画面全体がコンテンツ | (60, 60) 1800×900 |
| 中央に白板・ガラス板 | 板の内側（画像見て判断） |
| 3 分割で左上メイン | (40, 40) 1280×800 |
| 3 段パネル | 中央パネル内側 |

### サブコンテンツエリア（あるシーンのみ）

| 背景タイプ | 座標目安 |
|---|---|
| 右側縦長枠 | (1440, 80) 440×760 |
| 左上タイトル | (40, 40) 600×80 |
| タイトル札 | 背景の札位置に合わせる |

### 字幕エリア

| 背景タイプ | kind | 座標目安 |
|---|---|---|
| 下部白帯 | bar | (0, 960) 1920×120 |
| 下部黒帯 | bar | (0, 990) 1920×90 |
| 下部中央寄せ | bar | (180, 890) 1560×120 |
| 下部吹き出し | bubble | 画像に合わせて |
| 背景に帯なし（机） | bar (透明) or bubble | 適宜 |

### キャラスケール

| キャラ | 推奨 scale |
|---|---|
| 顔だけ小さく（両端下） | 0.50〜0.55 |
| 顔だけ中サイズ | 0.55〜0.60 |
| 体も見える（Scene 06 型） | 0.65〜0.70 |

## 共通の禁止配置

❌ 画像1 の水彩外枠をキャラが覆う
❌ 画像4 のタイトルラインの上にキャラが来る
❌ 画像6 のメカパネル中央（コンテンツ板）の上にキャラが乗る
❌ 画像8 のタイトル札にキャラが被る
❌ 画像12/19 のホワイトボード枠の上にキャラが乗る
❌ 画像14 の左右下部ガラス枠（キャラ用スロット）を字幕帯で塞ぐ
❌ 画像16/17 の上部タイトル帯の上にキャラが侵入
