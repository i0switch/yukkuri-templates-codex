# 寝る前スマホはなぜ眠りを壊すのか

## ツカミ（0:00–0:35）

**ずんだもん**：寝る前スマホは危険なのだ。

**めたん**：本当に怖いのは、ブルーライトだけじゃないのよ。

**ずんだもん**：え、スマホの光が悪い話じゃないのだ？

**めたん**：光も関係あるけど、もっと大きいのは習慣の罠ね。

[FIG:1] 寝る前スマホで脳が起きる全体像

**ずんだもん**：ボク、毎晩ちょっとだけ見てるのだ。

**めたん**：その「ちょっとだけ」が、一番あぶない入口なのよ。

**ずんだもん**：入口って聞くと、もう罠っぽいのだ。

**めたん**：そうね。今日の話は、根性論じゃなくて罠の外し方よ。

## セクション1：眠気は簡単に途切れる（0:35–1:10）

**めたん**：眠気は、電気みたいにスイッチで入るものじゃないの。

**ずんだもん**：待っていれば自然に眠くなると思ってたのだ。

**めたん**：実際は、刺激が少ない状態で少しずつ強くなる流れなのよ。

[INFO:1] 眠気の流れとスマホ刺激の関係

**ずんだもん**：じゃあスマホを見ると、その流れが止まるのだ？

**めたん**：そう。光、情報、感情がまとめて入ってくるからね。

**ずんだもん**：寝る前に脳へ仕事を投げてたのだ。

**めたん**：たとえば、寝ようとしている時に短い動画を見るとするでしょ。

**ずんだもん**：一本だけのつもりなのだ。

**めたん**：でも一本ごとに、笑う、驚く、比べる、考えるが起きるの。

**ずんだもん**：それはもう休憩じゃなくて作業なのだ。

**めたん**：しかも脳は、その刺激を「まだ起きる時間」と勘違いしやすいわ。

## セクション2：主犯は通知とスクロール（1:10–1:50）

**めたん**：一番やっかいなのは、通知とスクロールよ。

**ずんだもん**：ピコンって鳴るだけなのだ。

**めたん**：その一回で、脳は「確認しなきゃ」に戻るの。

[INFO:2] 通知から確認へ進む習慣ループ

**ずんだもん**：見る、反応する、また見るの流れなのだ。

**めたん**：そう。終わりが見えないから、時間の感覚も薄れるわ。

**ずんだもん**：気づいたら三十分消えてるやつなのだ。

[MAP:1] 光、通知、感情、習慣の関係図

**めたん**：ここで大事なのは、スマホが悪魔という話ではないことよ。

**ずんだもん**：スマホは便利なのだ。

**めたん**：便利だからこそ、寝る前には強すぎるの。

**ずんだもん**：甘いものを枕元に置く感じなのだ。

**めたん**：かなり近いわね。目に入るだけで、手が伸びやすくなるのよ。

## セクション3：意思より置き場所を変える（1:50–2:35）

**ずんだもん**：じゃあ根性で見なければいいのだ？

**めたん**：残念だけど、根性だけだとかなり弱いわ。

**ずんだもん**：ボク、もう負ける気がしてきたのだ。

**めたん**：だから戦う場所を変えるの。枕元に置かないのが第一歩ね。

[INFO:3] 枕元からスマホを離す対策図

**ずんだもん**：遠くで充電するのだ。

**めたん**：夜だけ通知を切って、朝に見る流れも作るといいわ。

**ずんだもん**：スマホを責めるより、仕組みを変えるのだ。

**めたん**：おすすめは、寝る九十分前に充電場所へ置くことね。

**ずんだもん**：九十分は長いのだ。

**めたん**：最初は三十分でもいいわ。大事なのは毎日できる距離よ。

**ずんだもん**：いきなり完璧を目指さないのだ。

**めたん**：そう。失敗しにくい配置を作る方が、ずっと現実的なの。

## まとめ（2:35–3:00）

**めたん**：まとめると、寝る前スマホの問題は光だけじゃないの。

**ずんだもん**：通知、スクロール、習慣が眠気を消すのだ。

**めたん**：完璧にやめるより、枕元から離す方が続きやすいわ。

[SLIDE:1] 光より習慣、意思より設計の3点まとめ

**ずんだもん**：まずは遠くで充電、通知オフ、朝に確認なのだ。

**めたん**：それだけでも、夜の入口はかなり変わるはずよ。

**ずんだもん**：ボクは今日から充電場所を変えるのだ。

**めたん**：今夜から、眠れる環境を先に作りましょう。

**ずんだもん**：睡眠を取り戻すのだ。今夜からなのだ。
