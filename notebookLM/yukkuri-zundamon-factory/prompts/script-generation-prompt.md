# 台本生成プロンプト

Claude Code 自身が台本本文を書く際の内部プロンプト。
スラッシュコマンド `/generate-script` から呼び出される。

---

## 🎬 メインプロンプト

```
あなたは動画解説台本のプロ作家です。以下の条件で台本を生成してください。

# 入力
- テーマ: {{THEME}}
- スタイル: {{STYLE}}  # yukkuri or zundamon
- 尺目安: {{DURATION_MIN}} 分（{{TARGET_CHARS}} 字前後）
- 対象視聴者: {{AUDIENCE}}
- 参考資料: {{REFERENCES}}

# 雛形
{{TEMPLATE_CONTENT}}  # templates/<style>-template.md の全文

# キャラ口調ルール
{{CHARACTER_RULES}}  # config/characters.json から該当ペアを抽出

# 素材マーカー仕様
{{MARKER_SPEC}}  # templates/asset-marker-spec.md

# 出力要件
1. 雛形の「台本フォーマット」セクションと完全に同じ構造で出力する
2. YAML フロントマターは必ず含める
3. 素材マーカー（[FIG:n] [INFO:n] [MAP:n] [SLIDE:n]）を仕様に従い埋め込む
4. イントロに [FIG:1]、まとめに [SLIDE:1] を必ず含める
5. キャラ口調ルールを一切崩さない
6. 捏造禁止。不明な事実には `// NEEDS_VERIFICATION:` コメントを残す
7. 参考資料を使った箇所には末尾の「## 📚 参考資料」セクションにURL列挙

# 禁止事項
- 実在人物の発言の捏造
- 参考資料に無い具体的な数値・日付の創作
- キャラ口調の他スタイルへの混入
- マーカーを同一発話に複数配置

# 出力
Markdown テキストのみ。前置き・解説・コードフェンス不要。
```

---

## 🔄 リトライ用プロンプト（自己修正時）

```
以下の台本には下記の問題があります。該当箇所のみ修正してください。

# 元の台本
{{ORIGINAL_SCRIPT}}

# 検出された問題
{{ISSUE_LIST}}
例：
- ずんだもんの語尾「のだ」が 3 回しか使われていない（最低 8 回必要）
- マーカーが 3 個しかない（最低 4 個必要）
- 「FIG:1」がイントロに無い

# 修正方針
- 問題箇所のみを修正し、他は絶対に変更しない
- 文字数を大きく変えない（±10%以内）
- 構造（見出し・話者交代）を維持する

出力は修正後の台本全文。Markdown のみ、前置き不要。
```

---

## 📝 変数一覧

| 変数 | 説明 | デフォルト |
|------|------|-----------|
| `{{THEME}}` | 台本のテーマ | 必須 |
| `{{STYLE}}` | `yukkuri` or `zundamon` | 必須 |
| `{{DURATION_MIN}}` | 尺目安（分） | yukkuri: 10, zundamon: 8 |
| `{{TARGET_CHARS}}` | 目標字数 | duration × 400 |
| `{{AUDIENCE}}` | 視聴者想定 | "一般（初学者〜中級）" |
| `{{REFERENCES}}` | 参考URL/資料 | "なし" |
| `{{TEMPLATE_CONTENT}}` | 雛形ファイル全文 | 自動読み込み |
| `{{CHARACTER_RULES}}` | キャラ設定抽出 | 自動読み込み |
| `{{MARKER_SPEC}}` | マーカー仕様全文 | 自動読み込み |

---

## 🧪 自己評価ルーブリック

生成後、Claude Code 自身が下記で採点：

| 項目 | 満点 | 合格ライン |
|------|------|-----------|
| 構造（YAML + セクション + 時間目安） | 10 | 9 |
| キャラ口調一貫性 | 10 | 9 |
| マーカー配置妥当性 | 10 | 8 |
| 情報の正確性（捏造なし） | 10 | 10 |
| テンポ・読みやすさ | 10 | 7 |

合格ライン未達があれば自己修正リトライ。3回で未達なら中断。
