このフォルダ全体を読み、以下の方針で作業してください。

目的:
- テーマから、ゆっくり解説台本 / ずんだもん＆めたん解説台本を生成できる状態を整える
- 台本から Scene ごとの素材要件を整理する
- NotebookLM を使って図解やスライドなどの素材を生成する運用を作る
- 生成素材の監査と差し戻しまで行えるようにする

やってほしいこと:
1. `README.md` と `CLAUDE.md` を読む
2. `workspace/projects/` 配下に案件が無ければ、私にテーマ候補を1つだけ聞く代わりに、既存テンプレから雛形案件を自動生成してよい
3. 雛形案件または指定案件について、以下の順に処理する
   - project_brief を整える
   - ゆっくり版台本を作る
   - ずんだもん＆めたん版台本を作る
   - Scene ごとの asset_manifest を作る
   - NotebookLM に投入するための素材生成メモを作る
   - 監査レポートの土台を作る
4. NotebookLM MCP が使えるか確認し、使えるなら MCP 前提の実行計画を出す
5. 使えないなら `nlm` CLI 前提の実行計画を出す
6. すべて `workspace/projects/<slug>/` に保存する

制約:
- Scene 番号を崩さない
- すべて日本語で管理する
- 進捗は短く伝える
- ファイルを作るだけでなく、実際に中身も埋める
