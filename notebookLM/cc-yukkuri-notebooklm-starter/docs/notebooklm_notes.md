# NotebookLM 連携メモ

## 現実的な使い方

NotebookLM で直接やるもの:
- 台本や参考資料の要約
- 概念図化
- スライド化
- 補助レポート生成

NotebookLM だけでは弱いもの:
- 動画用の厳密な「1カットごとの完成画像」
- 完全に指定通りのレイアウト再現
- キャラ立ち絵合成込みの最終フレーム完成

そのため、このスターターでは NotebookLM を「構造化された視覚素材の生成器」として使う。

## おすすめの生成方針

### infographic 向き
- 比較図
- 流れ図
- 因果関係
- 概念の全体像
- 数値のざっくり比較

### slide_deck 向き
- シーン単位の視覚素材候補
- 章ごとの導入
- 箇条書き中心の整理
- 図と短文の組み合わせ

### report 向き
- リサーチ補強
- シーンの事実確認
- 素材生成前の論点整理

## 生成前に作るべき要約
NotebookLM にそのまま台本全文だけを投げるのではなく、各 Scene ごとに以下をまとめた短い投入文を作るとズレが減る。

- この Scene の役割
- 視聴者に伝えたい一点
- 欲しい絵の種類
- 必須キーワード
- 避けたい誤解

## 監査で見るべきこと
- 主張がズレていないか
- 1枚で伝わるか
- 冗長でないか
- 台本のテンポを壊さないか
- その Scene に置く意味があるか
