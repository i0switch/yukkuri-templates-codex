# 解説台本プロンプト改善版

## 目的
このZIPは、テーマを1つ渡すだけで以下の台本を生成できるようにしたプロンプト集です。

- ゆっくり解説：霊夢 × 魔理沙
- ずんだもん解説：ずんだもん × 四国めたん

元プロンプトの良い点は残しつつ、以下を強化しています。

1. キャラ役割の固定
2. テーマだけ入力で成立する自動補完
3. 冒頭・中盤・終盤の視聴維持設計
4. バズりやすい企画化・タイトル化
5. 台本後の素材収集に使える指示
6. Claude Code / Codex どちらでも使える運用形式
7. 生成後の自己監査と改善ループ

## 使い方

### 最短入力
以下だけをプロンプト末尾に入れて実行してください。

```text
テーマ：〇〇
```

### おすすめ入力
```text
テーマ：〇〇
動画尺：3分
想定視聴者：初心者
方向性：面白さ重視
雰囲気：ややコミカル
必ず入れたい論点：〇〇、〇〇
避けたい表現：断定しすぎ、煽りすぎ
参考資料：URLやメモがあれば貼る
```

## ファイル構成

- `ゆっくり解説_霊夢魔理沙_台本生成プロンプト_改善版.md`
  - 霊夢と魔理沙の掛け合い台本用

- `ずんだもん解説_ずんだもんめたん_台本生成プロンプト_改善版.md`
  - ずんだもんと四国めたんの掛け合い台本用

- `共通_テーマだけで作るための入力テンプレ.md`
  - 毎回コピペする入力欄

- `台本監査_改善プロンプト.md`
  - 出てきた台本をチェックして改善する用

- `ClaudeCode_Codex用_実行指示.md`
  - Claude Code / Codex に一括作業させるための指示

## 改善ポイント

### 元プロンプトの良かったところ
- 霊夢・魔理沙、ずんだもん・めたんの役割分担はできている
- 出力形式が明確
- 冒頭フック、ミニオチ、自己監査が入っている

### 弱かったところ
- テーマだけ渡した場合、前提や角度が薄くなりやすい
- 「バズる企画」に変換する工程が弱い
- 中盤の離脱防止がまだ足りない
- 素材収集・動画化に使う情報が少し抽象的
- 生成後に改善ループへ回す指示が弱い

### 改善版で追加したもの
- 企画化フェーズ
- 視聴維持フック設計
- キャラ崩れ防止チェック
- 素材検索キーワード
- サムネ・タイトルの強化条件
- 台本スコアリング
- 70点未満なら自動修正するルール
