# 02_IMAGEGEN_PROMPT_PROMPT

## 目的

`script_final.md` の対象シーン全文をそのまま使って、ChatGPT Images 2.0などへ渡す `imagegen_prompt` を作る。

画像生成プロンプト本文は、対象シーン全文、固定プロンプト、台本に合わせた雰囲気指定だけにする。
`image_direction`、`visual_type`、`supports_dialogue`、`supports_moment` などの内部メタ情報は本文に入れない。

## 必須項目

- 対象シーンIDとタイトル
- 対象シーンの会話全文
- 固定プロンプト
- LLMが台本全体から判断した雰囲気指定

## 完成プロンプト形式

```yaml
imagegen_prompt: |-
  {scene_id}: {scene_title}

  {script_finalから対象シーン全文をそのまま貼る}

  ゆっくり解説動画向けの挿入画像を日本語で生成してください。 この画像は会話内容をそのまま再現するためのものではなく、シーンの要点・状況・概念・比喩を視覚的にわかりやすく補強するためのコンテンツ画像です。 字幕やセリフは別で表示するため、会話等は画像に入れないでください。 キャラクター同士の会話シーンにはせず、テーマ理解を助ける図解、アイコン、小物、UI、概念図、状況説明ビジュアルを中心に構成してください。 画面全体を有効活用し、情報が一目で伝わる、整理された高品質なビジュアルにしてください。 YouTubeの解説動画に適した、見やすく印象的で、内容理解を助ける16:9の横長構図で作成してください。 Make the aspect ratio 16:9.

  画像の雰囲気は{LLMが台本全体から判断した雰囲気}で生成してください。
```

## 注意

この固定プロンプト以外の構図、禁止リスト、品質基準、内部メタ情報を本文に足さない。
