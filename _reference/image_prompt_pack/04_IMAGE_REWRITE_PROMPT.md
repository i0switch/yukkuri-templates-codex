# 04_IMAGE_REWRITE_PROMPT

## 目的

生成前監査でFAILした画像計画を、同じプロンプトの微修正ではなく構図から直す。

## やること

- visual_typeを変える
- composition_typeを変える
- 会話内の補強ポイントを変える
- 前景/中景/背景を具体化する
- 必要なら「ボケ補強」から「誤解訂正」に変更する
- main/subの役割を分け直す
- Remotionで重ねる文字と画像内短語を分離する

## 禁止

- 同じプロンプトで再生成する
- 抽象アイコンを少し飾るだけで済ませる
- 白背景中央アイコンのまま色だけ変える
