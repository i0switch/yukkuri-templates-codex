# CHANGELOG_v3

## 追加した改善

- 最終行動の強度ルールを追加
  - 「見るだけ」「考えるだけ」で終わる台本を差し戻し
  - 確認、選択、保存/逃がす/テンプレ化、予定化から2アクション以上を要求
- 語尾バランスを強化
  - ZM: ずんだもん「のだ」「なのだ」20〜40%
  - RM: 魔理沙「だぜ」30〜60%
  - 0%/過剰/3連続を修正対象化
- 章タイトルのテーマ整合チェックを追加
  - フック型でも、テーマ外単語が混ざる場合は修正
- 自然文・誤字っぽい表現チェックを追加
  - 助詞抜け、口語違和感、キャラ語尾の衝突を修正対象化
- セルフ監査の減点強制ルールを追加
  - 90点以上やBlockingなし判定を甘く出さない
- 05_yaml_promptにYAML化前差し戻し条件を追加
  - 台本の弱点を抱えたままscript.yaml化しない

## 狙い

前回の改善で5分尺と掛け合いは改善されたが、以下が残った。

- ラストが「容量一覧を見る」だけで弱い
- 章タイトルにテーマ外の単語が混ざる可能性
- RM/ZMの語尾バランスがまだ監査しきれない
- 誤字ではないが違和感ある口語が残る
- セルフ監査が高得点に寄りすぎる

v3では、この最終品質ゲートを追加した。
