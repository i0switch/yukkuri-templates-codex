# 台本監査・改善プロンプト

以下の台本を、YouTube解説動画として本当に面白く、視聴維持されやすい内容になっているか監査してください。
対象は「ゆっくり解説」または「ずんだもん解説」です。

## 監査対象
```text
ここに生成済み台本を貼る
```

## 監査観点

### 1. キャラ構成
- ゆっくり解説の場合、霊夢が視聴者代表、魔理沙が解説役になっているか
- ずんだもん解説の場合、ずんだもんが疑問・感情役、めたんが解説・整理役になっているか
- 解説役が長文独演していないか
- リアクション役が相槌だけになっていないか
- 語尾や口調が崩れていないか

### 2. テーマだけで作れるか
- テーマの前提をAIが補完できているか
- 想定視聴者が明確か
- 結論が1文で言えるか
- 章立てが自然か
- 不明情報を捏造していないか

### 3. バズる可能性
- 冒頭5秒に意外性・損得・誤解訂正があるか
- タイトルとサムネがクリックしたくなるか
- 中盤に離脱防止の転換があるか
- 視聴者がコメントしたくなる問いがあるか
- 最後に人に話したくなる一言があるか

### 4. 分かりやすさ
- 専門用語が噛み砕かれているか
- 1セリフ1目的になっているか
- たとえ話が分かりやすいか
- 情報が詰まりすぎていないか
- 同じ内容の繰り返しがないか

### 5. 動画化しやすさ
- 画面枠メモがあるか
- 強調候補メモがあるか
- 必要素材リストが具体的か
- 検索キーワードが素材探しに使えるか
- BGM・SE・確定演出を台本段階で決めすぎていないか
- 最終演出は `02_演出編集プロンプト.md` に渡す前提になっているか
