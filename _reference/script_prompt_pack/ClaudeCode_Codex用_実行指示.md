# Claude Code / Codex用 実行指示

このフォルダは、Claude CodeまたはCodexのどちらでも使える台本生成プロンプト集です。
どちらか一方のエージェントで実行してください。

## 目的
テーマを渡すだけで、以下を生成する。

1. ゆっくり解説台本
2. ずんだもん解説台本
3. タイトル案
4. サムネ文言案
5. 必要素材リスト
6. 素材候補メモ
7. 強調候補メモ
8. 自己監査
9. 改善案

## 実行プロンプト

```text
このフォルダ内のプロンプトを読み込み、以下のテーマで台本を生成してください。

テーマ：ここにテーマを入れる
動画尺：3分
想定視聴者：初心者
方向性：面白さ重視
雰囲気：ややコミカル

実行内容：
1. `ゆっくり解説_霊夢魔理沙_台本生成プロンプト_改善版.md` を使って、霊夢と魔理沙の台本を生成する
2. `ずんだもん解説_ずんだもんめたん_台本生成プロンプト_改善版.md` を使って、ずんだもんとめたんの台本を生成する
3. それぞれ `outputs/` フォルダにMarkdownで保存する
4. 保存後、`台本監査_改善プロンプト.md` の観点で自己監査する
5. 70点未満の項目があれば、問題箇所だけ修正する
6. 最後に、どちらの台本が動画化しやすいか比較表を作る

注意：
- 霊夢・魔理沙、ずんだもん・めたんの役割を崩さない
- 冒頭は挨拶禁止。必ず意外性・損得・誤解訂正から入る
- 最新情報や数値が必要な場合は、根拠不明の断定を避ける
- 素材リストには検索キーワードを必ず入れる
- BGM、SE、確定演出、テロップ表示は台本段階で決めない
- Claude Codeの場合、画像生成はしない。素材候補・検索語・挿入位置まで作る
- Codexで画像生成スキルが使える場合でも、まずは素材仕様書として出す
```

## 出力ファイル例

```text
outputs/
  ゆっくり解説_テーマ名.md
  ずんだもん解説_テーマ名.md
  台本比較_テーマ名.md
  素材リスト_テーマ名.md
```
