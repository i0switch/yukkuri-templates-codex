# 改善内容メモ

## 元プロンプトのチェック結果

### キャラ構成
合格ライン。霊夢・魔理沙、ずんだもん・めたんの役割は分かれていた。
ただし、長いテーマや難しいテーマでは解説役が独演になりやすい可能性がある。

### バズる台本になるか
冒頭フックとミニオチの指定はあるため、最低限の設計はできていた。
ただし、タイトル・サムネ・中盤転換・コメント誘導が弱く、動画として伸ばす設計は追加した方がよい。

### テーマだけで作れるか
一応作れるが、テーマだけの場合にAIが「企画角度」を選ぶ工程が明示されていなかった。
そのため、普通の説明台本になりやすい。

## 改善版で追加したこと

- テーマ分解
- 企画角度の自動選定
- 冒頭フック案3本出し
- 中盤離脱防止ルール
- コメント誘導
- ショート切り抜き候補
- 素材検索キーワード
- 100点満点の自己監査
- 70点未満の部分修正ルール
- Claude Code / Codex用の実行指示
