# 事件解説型(章立て) 台本構成テンプレ

> ⚠️ **このテンプレを使う前に必ず「5. 倫理・法的配慮チェックリスト」を読むこと**。
> 実在の人物・事件を扱うため、推定無罪の原則・被害者遺族配慮・名誉毀損リスク等の検討が必須。
> 怠ると動画削除・チャンネル凍結・法的責任のリスクがある。

## 1. このアーキタイプの定義

「1つの事件・事案を多章構成で深掘りする、ドキュメンタリー寄り」の型。
オープニングで事件特定情報と複数の謎を畳みかけ、章立てで詳細を追い、エンディングで社会的教訓・問題提起で締める。
信頼性の柱は「一次ソース引用」と「推定無罪・配慮表現」。

判定の目安：
- 1つの実在の事件・事案を扱う（複数事件の比較は別アーキタイプ）
- 時系列に沿って章を立てられる
- 一次ソース（新聞・週刊誌・公的発表）が3本以上ある

## 2. 適するテーマ例

- 凶悪事件（殺人・行方不明・誘拐）
- 社会事件（食品偽装・データ改ざん・パワハラ）
- 歴史的事件（戦争・暗殺・大事故）
- 大企業の不祥事・スキャンダル
- 失踪・神隠し系の未解決事件
- 政治事件・冤罪事件

## 3. 基本構成テンプレ（章立て型）

### 3.1 オープニング（0:00〜1:30 目安）

**最強のフック設計**。次の4ステップを必ず含める：

1. **挨拶**：「こんにちは。ゆっくり霊夢です」「ゆっくり魔理沙だぜ」
2. **事件特定情報の即提示**：いつ・どこで・誰が（「20XX年X月、◯◯で起きた◯◯事件の◯◯について解説していくぜ」）
3. **複数の謎の畳みかけ**：3〜4個の矛盾・不可解点を立て続けに提示
   - 例：「◯◯と教述しながら事前に◯◯を検索していた矛盾」
   - 「◯◯と呼ばれた男が、なぜ◯◯したのか」
4. **章立て予告**：「事件の詳細な経緯から◯◯とは何者なのか、その生い立ちも含めてじっくり追っていこう」
5. **挨拶締め**：「それじゃあゆっくりしていってね」

### 3.2 各章（典型6〜8章、1章=3〜4分目安）

各章は次の5要素を必ず含む：

1. **章ヘッダ橋渡し**：「ここで重要なのが〜」「ここからが衝撃の事実だ」「天気が訪れたのは〜」型
2. **事実提示**：必ず一次ソース引用形式で提示
   - 「◯◯新聞によると〜」
   - 「校長は『〜』と謝罪している」
   - 「近隣住民の証言では〜」
   - 「FNN分析によれば〜」
3. **霊夢の感情リアクション**：共感的、視聴者代弁
   - 「3時間あったら助けられたかもしれないのに」
   - 「11歳でしっかりしてる」
   - 「まるで虐待」
4. **詳細・補足**：時系列の根拠、関係者証言、専門家の見解
5. **章末伏線**：「だが◯◯の裏では◯◯が同時に進行していたんだ」型で次章へ

橋渡し台詞のバリエーション例：
- 「ここが重要なんだ」「ここがこの事件の最も皮肉な構造の1つだ」「天気が訪れたのは〜」「だが◯◯の裏では〜」

### 3.3 章構成の典型パターン（行方不明・凶悪事件の場合）

```
第1章: 事件の全体像（時系列・場所・登場人物）
第2章: 消失/犯行の謎（時系列・現場検証）
第3章: 被害者の人物像（家庭環境・友人関係）
第4章: 容疑者の人物像（職場での評価・周囲の声）
第5章: 容疑者の生い立ち（過去の出来事）
第6章: 事件前後の不審行動（証拠・矛盾点）
第7章: 逮捕までの経緯（捜査の進展）
第8章: 社会的問い・教訓（背景にある社会問題）
```

事件の性質によって章を増減させる。原則は「時系列で章割り」。

### 3.4 エンディング（残り60秒目安）

固定要素：
1. 魔理沙「というわけで今回は◯◯について解説したぜ」（事件名回収）
2. 霊夢の事件総括（矛盾の指摘や疑問の言語化）
3. 魔理沙の社会的問い／教訓
   - 「この事件には現代社会が抱える◯◯と◯◯という問題が凝縮されていたんだ」
   - 「学校、地域、そして私たち1人1人が◯◯する必要があるぜ」
4. 霊夢「◯◯のご冥福を心からお祈りするとともに、この事件から得られる教訓を忘れてはいけないわね」
5. 「みんなの意見や感想があったら是非コメントしていって欲しいぜ」
6. 「高評価やチャンネル登録もよろしくお願いします」
7. 「最後までご視聴ありがとうございました」

ボケ・茶化しは皆無。品位を保って締める。

## 4. 対話パターン

| 役割 | 担当 | 比率 | 語尾 |
|---|---|---|---|
| 主任記者役 | 魔理沙 | 約6 | 〜だぜ／〜なんだ／〜と語っているぜ |
| 共感的聞き役・視聴者代弁 | 霊夢 | 約4 | 〜のね／〜かしら／〜じゃないの |

- 魔理沙は事実・証言・新聞引用を淡々と提示する記者役
- 霊夢は感情の窓口（「3時間あったら助けられたかもしれないのに」「11歳でしっかりしてる」「まるで虐待」）
- **ボケは皆無**。リスト型・サイエンス型と決定的に違うポイント
- 霊夢の感情リアクションは視聴者の倫理的反応を先回りして言語化する役割

## 5. 倫理・法的配慮チェックリスト ★最重要

このセクションは **台本生成前** と **生成後の監査時** の2回必ず確認する。

### 5.1 推定無罪の原則

- [ ] 容疑者・被告人を「犯人」と断定していない
- [ ] 「容疑者」「被告人」「とされる」「と教述している」等の表現を使っている
- [ ] 有罪確定後でない限り、有罪を前提とした表現を使わない
- [ ] 動画③の例：「鳴き柄を複数回移動させた詳しい経緯は今後の操作と裁判を待たなければならない。推定無罪の原則もあるし憶測で語ってはいけない」と明示パートを設けている

### 5.2 被害者・遺族配慮

- [ ] 被害者の実名扱いを慎重に判断（公表済みであっても、敬称・呼び方に配慮）
- [ ] 被害者の写真・動画の引用は最小限に
- [ ] 遺族のコメント・声明を引用する場合は出典を明示
- [ ] エンディングで「ご冥福をお祈り」型の追悼を必ず入れる
- [ ] 興味本位の表現（「グロ」「衝撃」連呼）を避ける

### 5.3 引用元の明示

- [ ] 章中の事実提示は必ず引用元を明示する（例：「◯◯新聞によると」「週刊◯◯の取材では」「警察発表によれば」）
- [ ] 出典は新聞・週刊誌・公的発表・本人SNS等の一次〜二次ソースに限る
- [ ] 動画概要欄に出典URLを明記する
- [ ] SNS上の匿名情報は「との噂もある」と明示し、確定事実として扱わない

### 5.4 名誉毀損・プライバシー侵害の予防

- [ ] 関係者（家族・友人・職場関係者）の実名は最小限に
- [ ] 国籍・民族・宗教等のセンシティブ属性は事件と直接関係する場合のみ言及
- [ ] 動画③の警告例：「SNS上では逮捕前の段階から特定の人物を名ざしで犯人と決めつける投稿や国籍に関する根拠のない投稿が広く拡散された」「冷静さを欠いたネット私刑は推定無罪の原則を踏みにじり冤罪のリスクや当事者の人権侵害という深刻な問題を孕んでいるぜ」
- [ ] テンプレに「ネット私刑への警告」セクションを入れることを推奨

### 5.5 センシティブ表現の自主規制

- [ ] 残虐描写・性的描写は最小限に
- [ ] 自殺事件の場合はWHOガイドライン準拠（手段の詳細を描写しない、原因を単純化しない）
- [ ] 未成年被害者の場合はさらに配慮を強化
- [ ] YouTubeのコミュニティガイドライン（センシティブコンテンツポリシー）を都度確認

### 5.6 動画概要欄に必須記載項目

```
本動画は◯◯の報道（◯◯新聞、週刊◯◯、◯◯テレビ等）を元に構成しています。
推定無罪の原則を尊重し、被害者・遺族の方々への配慮を心がけています。

主な参考資料：
- ◯◯新聞 20XX年X月X日付 https://...
- 週刊◯◯ 20XX年X月号
- ◯◯県警察発表 20XX年X月X日
（その他の出典）

ご遺族・関係者の皆様、視聴者の皆様で内容にご懸念のある方は
コメント欄またはチャンネルメッセージにてご連絡ください。
```

## 6. Scene-XX マッピング推奨

| 用途 | 推奨Sceneテンプレ | 補足 |
|---|---|---|
| オープニング | `Scene13`(暗3パネル) / `Scene16`(SF3スタックdark) | シリアス導入。明るい配色を避ける |
| 事件特定情報提示 | `Scene11`(白板UI) / `Scene04`(白板机) | 情報整理型 |
| 章本編（時系列・年表） | `Scene04`(白板机) / `Scene11`(白板UI) | 時系列を見せる |
| 章本編（証言引用） | `Scene17`(SF3スタックmid) / `Scene10`(モノ3パネル) | テキスト引用が見やすい |
| 場面転換 | `Scene09`(シネマバー) | 章間にBGM切替と組み合わせる |
| 教訓・社会問題 | `Scene15`(桜夜) / `Scene18`(山風景) | 締めの落ち着き |
| エンディング | `Scene15`(桜夜) | 追悼の品位 |

派手な装飾の Scene15（桜夜）以外の華やかなテンプレ（Scene20 虹ガラスなど）はエンディングでも避ける。

## 7. 長さ別目安表

| 動画尺 | 章数 | 1章あたり | 推奨用途 |
|---|---|---|---|
| 15分 | 5章 | 約2.5分 | 短時間の事件総まとめ |
| 20分 | 6章 | 約3分 | 標準版 |
| 25分 | 7章 | 約3.3分 | 標準版（参考動画③26:50に近い） |
| 30分 | 8章 | 約3.5分 | 長尺・大事件 |

シーン数の目安：1章=3〜4 scene、オープニング2 scene、エンディング1 scene。
- 25分版なら 2+7×3+1 = 24 scene 前後

## 8. 実例引用

### 動画③ 「【ゆっくり解説】安達優季の正体が…京都南丹市小5行方不明事件のその後」
- チャンネル：ゆっくり凶悪事件簿
- 出典：https://www.youtube.com/watch?v=uutCkxbz0FQ
- 尺：26:50

⚠️ 以下の引用は **構成テンプレを示すための型の例示** であり、特定事件への評価・推測を含意するものではない。実際にこの事件を扱う動画を作る場合は一次ソースを直接参照すること。

#### オープニング（事件特定情報＋複数の謎の畳みかけ）

> 霊夢「こんにちは。ゆっくり霊夢です。」
> 魔理沙「ゆっくり魔理沙だぜ。今回は2026年3月に起きた京都市〇〇事件の犯人◯◯について解説していくぜ。」
> 霊夢「あの卒業式の朝に◯◯君を手にかけた義父ね。」
> 魔理沙「中学時代は生徒会長、20年金属の品質管理課長。上司からは温厚、人から好かれると評されていた男だが、その大人しい義父の裏にはいくつもの顔があったんだ。2度の結婚と破綻した家庭、子が漏らした『変なおっさん』という呼び名。そして衝動的に犯行に及んだと供述しながら事前にドラレコの消し方を検索していた矛盾。」
> 霊夢「穏やかな会社員の顔と11歳の命を奪った顔。一体どちらが本当の姿なの？」
> 魔理沙「それじゃあ事件の詳細な経緯から◯◯とは何者なのか。その生い立ちも含めてじっくり追っていこう。それじゃあゆっくりしていってね。」

実装ポイント：
- 開始15秒以内に「いつ・どこで・誰が」を提示
- 30秒以内に複数の謎を3つ以上畳みかけ
- 「一体どちらが本当の姿なの？」型の問いで吊る
- 「経緯から◯◯まで追う」と章立てを予告

#### 章ヘッダ橋渡しの典型例

> 魔理沙「ここが重要なんだ。普段の◯◯は祖母が担当していたとされている。この日に限って義父が送迎したというんだぜ。」

> 魔理沙「だが、神隠しのように見えたこの消失の裏では、◯◯容疑者の周到な計算と、そしてバレないための混乱が同時に進行していたんだ。」

実装ポイント：
- 章ヘッダは「ここが重要なんだ」「だが◯◯の裏では〜」型
- 章末で次章への伏線を必ず張る

#### 一次ソース引用の典型例

> 魔理沙「校長は『8時半の段階で欠席が分かっていたのにご家庭にご連絡するのが遅くなった。たとえ式であってもきちんと確認すべきだった』と謝罪している。南丹市教育委員会も『欠席確認が不十分だった。深くお詫びする』と述べているぜ。」

> 魔理沙「消防団長はこう語っているぜ。『発見されたところはもちろん捜索対象になっていて何回も確認、捜索はしたはずだが…』」

実装ポイント：
- 引用元（校長／教育委員会／消防団長）と発言を明示
- 直接引用は鉤括弧で囲み、要約部分と区別する

#### 推定無罪の警告パート（事件型では必須）

> 魔理沙「鳴き柄を複数回移動させた詳しい経緯は、今後の捜査と裁判を待たなければならない。」
> 霊夢「推定無罪の原則もあるし、憶測で語ってはいけないこともあるよね。」
> 魔理沙「SNS上では逮捕前の段階から特定の人物を名ざしで犯人と決めつける投稿や、国籍に関する根拠のない投稿が広く拡散された。関西テレビでは別人の映像を容疑者として誤って放送する事態も起きている。冷静さを欠いたネット私刑は推定無罪の原則を踏みにじり、冤罪のリスクや当事者の人権侵害という深刻な問題を孕んでいるぜ。」
> 霊夢「事件に関心を持つことと、誰かを断罪することは別問題よね。」
> 魔理沙「その通りだ。事件の全容が明らかになるまで、冷静に捜査と裁判の行方を見守っていくことが、私たち視聴者にもできる最低限のことだぜ。」

実装ポイント：
- このパートは **必須**。事件型のテンプレを使う以上、必ず1パート設ける
- ネット私刑への警告と推定無罪の説明をセット
- 視聴者への呼びかけ（「私たち視聴者にもできる最低限のこと」）で締める

#### エンディング（教訓・社会問題化）

> 魔理沙「というわけで今回は◯◯容疑者について解説したぜ。」
> 霊夢「卒業式の朝に消えた11歳。衝動的にと供述されているけど事前にドラレコの消し方を調べていたりと、矛盾が多すぎる事件だったわ。」
> 魔理沙「再婚から3ヶ月、4世代同居の休暇に婿入りした大人しいはずの義父。そしてパパの話はやめてと友人に漏らしていた少年。この事件には現代社会が抱える『ステップファミリーの困難』と『見えない虐待』という問題が凝縮されていたんだ。子供の何気ない一言を大人がどう受け止めるか。それが時として小さな命を守る最後のサインになる。学校、地域、そして私たち1人1人が子供のSOSを拾える感度を持ち続ける必要があるぜ。」
> 霊夢「◯◯君のご冥福を心からお祈りするとともに、この事件から得られる教訓を忘れてはいけないわね。」
> 魔理沙「みんなの意見や感想があったら是非コメントしていって欲しいぜ。」
> 霊夢「高評価やチャンネル登録もよろしくお願いします。」
> 魔理沙「では今回はここまで。」
> 霊夢「最後までご視聴ありがとうございました。」

実装ポイント：
- 事件の矛盾の総括 → 社会問題のキーワード抽出（「ステップファミリーの困難」「見えない虐待」）→ 視聴者への呼びかけ → 追悼 → コメント誘導 → 定型挨拶
- 興味本位の煽りで終わらず、社会派の品位で締める

## 9. 典型的な失敗パターン

| 失敗 | 症状 | 防ぎ方 |
|---|---|---|
| 推定無罪表現抜け | 「犯人は〜」と断定 | 「容疑者」「とされる」「と供述している」を全文一括チェック |
| センセーショナル過剰 | 「ヤバすぎ」「闇深」連呼 | 1動画あたり煽り表現を5回以下に制限 |
| 章長アンバランス | 第1章10分、第7章2分など | 各章でセリフ数の目標を設定（例：14〜18セリフ） |
| 情報源未明示 | 「らしい」「と言われる」だけで根拠不明 | 各章に最低1つの一次ソース引用を必須化 |
| 推定無罪パート抜け | 警告セクションがない | テンプレ準拠チェックで必ず確認 |
| エンディング軽薄化 | 「次回もお楽しみに」だけで終わる | 「教訓→ご冥福→コメント→挨拶」の4段を必ず |
| 字幕25文字超 | 1セリフが長すぎ画面に入らない | `dialogue[].text` を25文字以内に分割 |
| キャラ崩れ | 霊夢がボケる | 事件型ではボケ禁止。監査プロンプトで毎回チェック |

## 10. テンプレ使用時の事前準備チェックリスト

台本生成前に以下を揃える：

- [ ] **一次ソース3本以上**（新聞・週刊誌・公的発表）と各URL
- [ ] **時系列年表**（事件発生〜現在までの主要イベント）
- [ ] **関係者相関図**（被害者・容疑者・周辺人物）
- [ ] **論点メモ**（謎・矛盾点・社会的問題のリスト）
- [ ] **引用予定の発言**（出典＋発言内容を表形式で）
- [ ] **配慮検討事項**（被害者属性・遺族の意向・継続中の捜査の有無）
- [ ] **動画公開タイミング**（係争中なら表現を強めない）

## 11. このテンプレで台本生成する際の入力チェックリスト

```text
アーキタイプ：事件解説型(章立て)
事件名：（例：◯◯事件）
発生日時・場所：
```

## 12. 公開後の対応

事件解説型は他のアーキタイプより炎上・苦情のリスクが高い。公開後は以下を運用する：

- コメント欄で関係者・遺族からの連絡があった場合、24時間以内に対応
- 事実誤認の指摘があった場合、概要欄に訂正を追記する
- 続報（裁判結果等）が出た場合、続編動画 or 概要欄追記でフォロー
- YouTubeから警告が来た場合、即座に該当箇所を非公開・編集
