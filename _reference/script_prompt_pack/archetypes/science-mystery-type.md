# サイエンス問題提起型 台本構成テンプレ

## 1. このアーキタイプの定義

「1つの大きな問いを立てて、仮説→反証→再仮説 を繰り返しながら議論を展開する」型。
30分前後の長尺が前提で、章ごとに小問題を解決しながら全体の大問題に迫る。
学術正確性とエンタメ性のバランス、ボケ密度の管理が肝。

判定の目安：
- 「なぜ◯◯なのか？」「◯◯は本当に◯◯なのか？」型の問いが立つ
- 結論が完全には確定していない（仮説が複数ある）
- 一次論文 or 教科書レベルの参考資料がある

## 2. 適するテーマ例

- 進化のミステリー（人類進化、生物進化、絶滅）
- 宇宙・天文学（ダークマター、ブラックホール、地球外生命）
- 古代史・考古学（文明の起源、遺跡の謎）
- 脳科学・心理学（意識、記憶、選択行動）
- 経済学パラドクス（合理性、行動経済学）
- 物理学パラドクス（時間、量子力学）

## 3. 基本構成テンプレ

### 3.1 オープニング（0:00〜2:00 目安）

サイエンス型の最大の特徴は **学術導入フック → 寸劇ボケで日常接続 → 本題提示** の3段構造。

1. **挨拶**：「ゆっくり霊夢です」「ゆっくり魔理沙だぜ」
2. **学術導入フック**（0:30〜1:00）：
   - 意外な実験結果、統計、パラドクス、矛盾を1つ提示
   - 例：「ジャム24種類より6種類の方が売上10倍」（選択のパラドクス）
   - 視聴者に「なんでだろう？」と引っかかりを作る
3. **寸劇ボケで日常接続**（5〜10秒）：
   - 学術ネタを日常の小ネタにつなげて緊張を緩める
   - 例：「というわけで魔理沙の借金を肩代わりする気はない」
4. **本題提示**：
   - 「今回はなぜか1種類しかない人類のミステリー、なぜ人類は1種類だけになったのかについて解説していくぞ」
   - 学術導入フックと本題の接続を匂わせる（伏線として終盤で回収）
5. **挨拶締め**：「ゆっくりしていってね」

### 3.2 章構成（典型5章、1章=5〜7分目安）

各章は次の流れで組み立てる：

1. **章ヘッダ**：問いの言語化（「人類は元々多様だった」「なぜ他の人類は消えたのか」）
2. **問題提起**：霊夢が素朴な疑問を出す（「え、私たち以外にも人類がいたなんてとても信じられないわね」）
3. **証拠提示**：化石・実験データ・歴史的記録を魔理沙が提示
4. **仮説提示**：複数の仮説を順に並べる（「説1：気候変動説」「説2：認知能力説」）
5. **反証・補強**：各仮説に対する反論や補強データ
6. **再仮説 or 暫定結論**：「現時点では◯◯説が有力とされている」
7. **章末予告**：「次は◯◯について見ていくぜ」で次章へ

### 3.3 章構成の典型パターン（ミステリー型サイエンス）

```
第1章: 問題提起と現状確認（「なぜ◯◯なのか」と「現状は◯◯」）
第2章: 過去の事実・前提知識（「もともとは◯◯だった」）
第3章: 候補となる答え・登場人物の紹介（「これまでに存在した◯◯一覧」）
第4章: 主要な仮説と検証（仮説1〜N、反証含む）
第5章: 暫定結論と未来予測（「◯◯が有力／今後はどうなるか」）
```

### 3.4 エンディング（残り90秒目安）

固定要素：
1. **学術的締め**：暫定結論 or 哲学的問い
   - 例：「答えが出るのはまだ先になりそうだな。ということで今回はなぜ人類は1種類だけになったのかについて解説してきたぜ」
2. **冒頭フック回収**：オープニングの学術導入フックを終盤で回収
   - 例：「たった1種類残った人類、奇跡のように与えられた命を私たちは大切にしないといけないわね」
3. **ボケ締め**（任意だが推奨）：オープニングの寸劇ボケと対になる二段締め
   - 例：「魔理沙も最も身近な命である私が植えるのを防ぐためにぜひとも借金の肩代わりの協力を」「寝る間も死んで働けの一択だな」
4. **定型挨拶**：「動画が面白かったら高評価とチャンネル登録よろしくお願いします」

## 4. 対話パターン

| 役割 | 担当 | 比率 | 語尾 |
|---|---|---|---|
| 学者役・解説 | 魔理沙 | 約5 | 〜だぜ／〜と考えられているぜ／〜なんだ |
| 好奇心強い生徒役・ボケ | 霊夢 | 約5 | 〜なのね／〜じゃない／〜だってばよ（崩しOK） |

- 魔理沙は学術用語を噛み砕いて解説する学者役
- 霊夢は **ボケ多め**（4本の参考動画中、ボケ密度は最多）
- 霊夢の語尾の崩し（「だってばよ」「〜的な？」「〜よね？」等）も許容
- 比率は5:5でリスト型・事件型より霊夢の比重が大きい

## 5. 長尺退屈防止の3原則 ★最重要

サイエンス型は30分前後の長尺になるため、退屈防止の設計が必須。
次の3原則を **必ず** テンプレに組み込む。

### 5.1 ボケ密度：1章に最低1回の脱線ボケ

霊夢のキャラ立ち台詞を章ごとに最低1回入れる。
動画④の例：

- 「人類には2種類の人間がいるわ。自ら困難に挑戦し成功を収める人間と、ただに過ごして…」→「そういうことじゃねえよ」
- 「ハイブリッドカーみたいな」
- 「装飾系男子」
- 「私も空くらい飛べるし」型の自己ツッコミ

ボケパターン：
- **的外れ回答型**：問いに対してすっとぼけた答えをする
- **比喩誤用型**：日常の物事に強引に喩える
- **自己アピール型**：「私だってできるし」と張り合う

### 5.2 章間予告：「次は〜について見ていくぜ」の橋渡し必須

長尺で離脱されないために、各章末に必ず次章への予告を入れる。
- 「次は◯◯について見ていくぜ」
- 「ここからは◯◯について話していこう」
- 「それでは具体的に◯◯を見ていくぞ」

予告がない章末は離脱ポイントになる。

### 5.3 冒頭フック回収：終盤で伏線を回収

オープニングの学術導入フックを、終盤で必ず回収する。
- 動画④の例：冒頭「ジャム実験（選択肢が多いと売上低下）」→ 終盤「たった1種類残った人類、奇跡のように与えられた命」で「1種類だけ残った」と「選ばれた1つ」を響かせる
- 視聴者に「最後まで見てよかった」と思わせる構造

回収忘れがあるとフックが浮いた状態になる。台本生成時に「冒頭フックの回収位置」を必ずマークする。

## 6. Scene-XX マッピング推奨

| 用途 | 推奨Sceneテンプレ | 補足 |
|---|---|---|
| 学術導入フック | `Scene07`(ラボガラス) / `Scene06`(メカパネル) | 学術感のある画面 |
| 寸劇ボケ | `Scene12`(教室バブル) / `Scene08`(和室札) | 日常感のある画面 |
| 本題提示・問題提起 | `Scene01`(水彩外枠) / `Scene19`(白板机v2) | タイトル枠あり |
| 章本編・仮説提示 | `Scene02`(グレー3パネル) / `Scene10`(モノ3パネル) / `Scene11`(白板UI) | 図解・比較に強い |
| 証拠・データ提示 | `Scene04`(白板机) / `Scene11`(白板UI) | 情報整理型 |
| 場面転換 | `Scene09`(シネマバー) / `Scene14`(マルチレイヤーガラス) | 章間の切替 |
| 哲学的締め | `Scene15`(桜夜) / `Scene18`(山風景) | 余韻のある締め |

テンプレートは動画全体で1つだけ選び、`meta.layout_template` に指定する。章やボケパートでテンプレートを切り替えず、同じ見た目スキンの中で `main` / `sub` / `title_text` / `dialogue` の内容だけを変える。

## 7. 長さ別目安表

| 動画尺 | 章数 | 1章あたり | 推奨用途 |
|---|---|---|---|
| 15分 | 3章 | 約4.5分 | 短時間版（中堅サイエンスチャンネル向け） |
| 20分 | 4章 | 約4.5分 | 標準版 |
| 30分 | 5章 | 約5.5分 | 標準版（参考動画④32:02に近い） |
| 40分 | 6章 | 約6分 | 大型解説 |

シーン数の目安：1章=5〜7 scene、オープニング3 scene（フック・ボケ・本題提示）、エンディング2 scene。
- 30分版なら 3+5×6+2 = 35 scene 前後

## 8. 実例引用

### 動画④ 「【ゆっくり解説】消えた人類の謎…なぜ人類は1種類だけになったのか？」
- チャンネル：世界の未解明ミステリー【ゆっくり解説】
- 出典：https://www.youtube.com/watch?v=_11OxkwLBTI
- 尺：32:02

#### オープニング（学術導入フック → 寸劇ボケ → 本題提示の3段構造）

> 霊夢「ゆっくり霊夢です。」
> 魔理沙「ゆっくり魔理沙だぜ。スーパーマーケットでお客にジャムの試食をさせる実験をしたんだけど、24種類のジャムが売られる場合と6種類のジャムが売られる場合では、6種類の方が売上が10倍も伸びたんですって。つまり人間は選択肢が多すぎると無意識に選択自体を避けてしまうから、選択肢は少なくした方がいいというわけよ。」
> 霊夢「なるほどな。」
> 魔理沙「というわけで魔理沙の借金を肩代わりする気はない。はい。お、いいえ。」
> 霊夢「というわけで商品点数を多くしすぎるとマーケティング的には不利になるわけだが、今回はなぜか1種類しかない人類のミステリー、なぜ人類は1種類だけになったのかについて解説していくぞ。ゆっくりしていってね。」

実装ポイント：
- 学術導入フック「ジャム実験」で意外性を提示
- 寸劇ボケ「借金肩代わり」で日常接続＋緊張緩和（後で回収する伏線）
- 「なぜか1種類しかない」で本題と冒頭フックを論理的に接続

#### ボケパートの典型（章内のキャラ立て）

> 魔理沙「じゃあ人類は？」
> 霊夢「人類には2種類の人間がいるわ。自ら困難に挑戦し、成功を収める人間と、ただに過ごして…」
> 魔理沙「そういうことじゃねえよ。人類、いわゆる人というものは現在の地球にホモサピエンスの1種類しかいない。」
> 霊夢「それはそうじゃない。だって人は人でしょ。」

実装ポイント：
- 「2種類の人間がいる」というネット定型ボケを霊夢に振らせる
- 魔理沙が「そういうことじゃねえよ」とツッコミで本題に戻す
- ボケ → ツッコミ → 本題復帰 の3段で章のリズムを作る

#### 章ヘッダ橋渡し

> 魔理沙「それに関しては今も研究の続く大テーマなんだ。今回はどうして私たち人類はホモサピエンスの1種類しかいないのかという謎に迫っていきたいと思うぜ。」
> 「人類は元々多様だった。今説明したように人か人族というのはホモサピエンスの一しかいない。」
> 霊夢「そうだったわね。」
> 魔理沙「しかしかつてはそうではなかったんだ。」

実装ポイント：
- 章ヘッダは前章の結論を1行で受けてから新しい論点を提示する
- 「今説明したように〜。しかし〜」型の接続で論理の連続性を担保

#### 仮説提示と反証の流れ

> 魔理沙「まずは有名なネアンデルタール人だ。」
> 魔理沙「うっほっほ。そういう言い方はネアンデルタール人差別だぜ。というのもネアンデルタール人は一昔前には野蛮な原始人として語られることが多かった。」
> 霊夢「確かにそういうイメージあったわ。」
> 魔理沙「ところがそこが大きな誤解なんだ。彼らの脳のサイズは現生人類と同等かそれ以上に大きかったと言われているぜ。」
> 霊夢「え、そうなの？それなのに滅びちゃったの？」

実装ポイント：
- 「俗説 → 反証 → 新事実 → 新たな疑問」の4段で展開
- 霊夢の疑問が次の章への橋渡しになる

#### 終盤の伏線回収＋ボケ締め

> 魔理沙「こうした技術によって人類は再び複数の種に別れていくのか。そしてそれは本当に幸せな進化なのか。答えが出るのはまだ先になりそうだな。ということで今回はなぜ人類は1種類だけになったのかについて解説してきたぜ。」
> 霊夢「たった1種類残った人類、奇跡のように与えられた命を私たちは大切にしないといけないわね。魔理沙も最も身近な命である私が植えるのを防ぐためにぜひとも借金の肩代わりの協力を」
> 魔理沙「寝る間も死んで働けの一択だな。というわけで今日の動画はここまで。動画が面白かったら高評価とチャンネル登録よろしくお願いします。」

実装ポイント：
- 「答えが出るのはまだ先」で哲学的締め
- 「奇跡のように与えられた命」で冒頭フック「ジャム実験＝選ばれた1つ」を間接的に回収
- 「借金の肩代わり」で冒頭ボケを回収する二段締め
- 真面目な締め＋ボケ締めのコンボで余韻を残す

## 9. 典型的な失敗パターン

| 失敗 | 症状 | 防ぎ方 |
|---|---|---|
| 学術濃度高すぎ | 専門用語連発で離脱多発 | 1章につき新出専門用語は3つまで、必ず日常の喩えで補う |
| ボケ皆無で説教臭 | 30分間ずっと真面目で離脱 | ボケ密度3原則の §5.1 を遵守 |
| 章間予告抜け | 章末で「以上です」とだけ言う | 各章末に「次は〜」を必ず入れる §5.2 |
| 伏線回収忘れ | 冒頭フックが浮いたまま終わる | 台本作成時に「冒頭フックの回収位置」をマーク §5.3 |
| 仮説提示の論理矛盾 | 仮説Aと仮説Bが同じことを言っている | 仮説は対立構造を明確に |
| 字幕25文字超 | 1セリフが長すぎ画面に入らない | `dialogue[].text` を25文字以内に分割 |
| キャラ崩れ | 霊夢が完全にツッコミ役になる | 霊夢のボケを章ごとに最低1回確保 |
| 反証なしの単一仮説 | 1つの説だけ紹介して終わる | 必ず複数仮説と反証を提示 |

## 10. テンプレ使用時の事前準備チェックリスト

台本生成前に以下を揃える：

- [ ] **一次論文 or 教科書1冊以上**（一般書籍でも可、ただし学術的に信頼できるもの）
- [ ] **対立する仮説 or 反証データ**を最低2つ
- [ ] **読者の前提知識ライン**設定（中学レベル/高校レベル/大学レベル）
- [ ] **冒頭フックの題材**（意外な実験・統計・パラドクス）
- [ ] **冒頭フックと本題の接続点**（メタファー or 共通テーマ）
- [ ] **冒頭フック回収用の終盤台詞**（事前にマーク）
- [ ] **章ごとのボケ案**（1章につき最低1ボケ）
