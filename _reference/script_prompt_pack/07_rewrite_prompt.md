# 07_rewrite_prompt

## 目的

監査FAIL箇所だけを直す。台本全体をむやみに作り直さない。

## 優先順位

1. 尺不足、シーン不足、セリフ不足。
2. 聞き役化、Q&A化、独演。
3. 章タイトルの説明目次化。
4. 数字、具体例、あるある不足。
5. リアクション強度不足。
6. 口癖の偏り。
7. テンプレート枠との不整合。
8. 画像プロンプトとの不整合。
9. 最終行動の弱さ。

## ルール

- 問題のないシーンは維持する。
- 解説を増やすより、掛け合いの温度差を増やす。
- 解説役の独演は視聴者代表の反応で分割する。
- 画像プロンプトを直す場合は、先に `script_final.md` の対象シーン本文と作風キーワード（数語）を直す。中間ファイル `image_direction` には依存しない。
- 抽象プロンプトへ戻さない。`hook_type` / `visual_type` / `composition_type` / `supports_*` を `imagegen_prompt` 本文に混ぜない。

## 出力形式

```md
# 台本修正版

## 1. 修正概要
| 修正対象 | 修正理由 | 対応 |
|---|---|---|

## 2. 修正後の該当箇所

### sXX: フック型タイトル
修正後のシーン全文。

## 3. 修正後セルフ監査
- Blocking Issues 解消:
- 尺と密度:
- キャラ口調:
- 独演なし:
- テンプレート適合:
- 画像プロンプト適合:
- 軽微な改善余地:
- 判定:
```
