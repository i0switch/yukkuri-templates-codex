# 09_image_prompt_audit

## 目的

直投げ型では、画像プロンプト監査を必須ゲートにしない。

このファイルは、任意で人間確認するときの軽い観点だけを残す。

## 任意確認

- `script_final.md` の対象シーン全文が入っている。
- アスペクト比16:9が明記されている。
- 会話は別で字幕表示する前提が書かれている。
- 画像はコンテンツ部分の挿入画像として指定されている。
- LLMが台本全体から作風を判断している。
- 会話全文を画像内文字として並べる指示になっていない。

## 出力形式

必要な場合だけ残す。

```md
# 画像プロンプト任意確認

- 判定: PASS / FAIL
- 確認内容:
- 修正したい作風:
```

この確認が無いことを理由に `pre-render-gate` や `build-episode` を止めない。
