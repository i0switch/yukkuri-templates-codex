# 08 Image Generation v2

## 目的

`script_final.md` の対象シーン全文と固定プロンプトだけを使い、ゆっくり解説 / ずんだもん解説動画のコンテンツ枠に挿入する16:9画像プロンプトを作る。

会話は別途字幕で出すため、画像は会話文を説明カード化しない。プロンプト本文には、対象シーン全文、固定プロンプト、台本に合わせた雰囲気指定以外を入れない。

## 入力

- episode全体のテーマ
- `script_final.md` 全体
- 対象シーンID
- 対象シーンタイトル
- 対象シーンの会話全文
- 台本全体から判断した画像の雰囲気

## 作風指定

LLMが `script_final.md` 全体を読み、題材に合う作風を決めてプロンプト末尾に追加する。

例:

- 怪談解説: 暗い照明、影、湿度感、不穏、低彩度
- 医薬品解説: 清潔な病院、白、淡い青、安心感、整理された画面
- 通知/スマホ依存: 生活空間、スマホ通知、疲労感、時間が溶ける表現
- 節約/お金: レシート、家計簿、明るい生活感、損失回避

## 画像内テキスト

固定プロンプト内で「字幕やセリフは別で表示するため、会話等は画像に入れない」と明記する。
追加の禁止リスト、構図指定、内部メタ情報は画像生成プロンプト本文に足さない。

## 出力形式

```yaml
visual_asset_plan:
  - slot: main
    purpose: "script_final直投げ型の挿入画像"
    imagegen_prompt: |-
      s03: 通知は生活のすき間を全部食う

      霊夢「...」
      魔理沙「...」

      ゆっくり解説動画向けの挿入画像を日本語で生成してください。 この画像は会話内容をそのまま再現するためのものではなく、シーンの要点・状況・概念・比喩を視覚的にわかりやすく補強するためのコンテンツ画像です。 字幕やセリフは別で表示するため、会話等は画像に入れないでください。 キャラクター同士の会話シーンにはせず、テーマ理解を助ける図解、アイコン、小物、UI、概念図、状況説明ビジュアルを中心に構成してください。 画面全体を有効活用し、情報が一目で伝わる、整理された高品質なビジュアルにしてください。 YouTubeの解説動画に適した、見やすく印象的で、内容理解を助ける16:9の横長構図で作成してください。 Make the aspect ratio 16:9.

      画像の雰囲気は生活空間、スマホ通知、疲労感、時間が溶ける表現で生成してください。
```

## 重要

- `script.yaml` 内の `visual_asset_plan[].imagegen_prompt` を正本にする。
- `image_direction`、`visual_type`、`supports_dialogue`、`supports_moment` は必須ではない。
- `imagegen_prompt` 本文に `visual_type`、`composition_type`、`supports_moment`、`scene_template` などの内部メタ情報を入れない。
- `remotion_card_plan.md` は生成しない。
- `main.caption` / `sub.caption` は使わない。
